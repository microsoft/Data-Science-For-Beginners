**‡∞ï‡∞æ‡∞≤‡∞Æ‡±ç ‡∞é‡∞Ç‡∞™‡∞ø‡∞ï**. ‡∞Æ‡±á‡∞Æ‡±Å ‡∞µ‡±ç‡∞Ø‡∞ï‡±ç‡∞§‡∞ø‡∞ó‡∞§ ‡∞ï‡∞æ‡∞≤‡∞Æ‡±ç‚Äå‡∞≤‡∞®‡±Å `df['A']` ‡∞Ö‡∞®‡∞ø ‡∞∞‡∞æ‡∞Ø‡∞°‡∞Ç ‡∞¶‡±ç‡∞µ‡∞æ‡∞∞‡∞æ ‡∞é‡∞Ç‡∞ö‡±Å‡∞ï‡±ã‡∞µ‡∞ö‡±ç‡∞ö‡±Å - ‡∞à ‡∞Ü‡∞™‡∞∞‡±á‡∞∑‡∞®‡±ç ‡∞í‡∞ï ‡∞∏‡∞ø‡∞∞‡±Ä‡∞∏‡±ç‚Äå‡∞®‡±Å ‡∞á‡∞∏‡±ç‡∞§‡±Å‡∞Ç‡∞¶‡∞ø. ‡∞Æ‡±á‡∞Æ‡±Å `df[['B','A']]` ‡∞Ö‡∞®‡∞ø ‡∞∞‡∞æ‡∞Ø‡∞°‡∞Ç ‡∞¶‡±ç‡∞µ‡∞æ‡∞∞‡∞æ ‡∞ï‡∞æ‡∞≤‡∞Æ‡±ç‚Äå‡∞≤ ‡∞â‡∞™‡∞∏‡∞Æ‡∞ø‡∞§‡∞ø‡∞®‡∞ø ‡∞Æ‡∞∞‡±ã ‡∞°‡±á‡∞ü‡∞æ‡∞´‡±ç‡∞∞‡±á‡∞Æ‡±ç‚Äå‡∞ó‡∞æ ‡∞é‡∞Ç‡∞ö‡±Å‡∞ï‡±ã‡∞µ‡∞ö‡±ç‡∞ö‡±Å - ‡∞á‡∞¶‡∞ø ‡∞Æ‡∞∞‡±ã ‡∞°‡±á‡∞ü‡∞æ‡∞´‡±ç‡∞∞‡±á‡∞Æ‡±ç ‡∞á‡∞∏‡±ç‡∞§‡±Å‡∞Ç‡∞¶‡∞ø.

**‡∞®‡∞ø‡∞∞‡±ç‡∞¶‡∞ø‡∞∑‡±ç‡∞ü ‡∞µ‡∞∞‡±Å‡∞∏‡∞≤‡∞®‡±Å ‡∞´‡∞ø‡∞≤‡±ç‡∞ü‡∞∞‡±ç ‡∞ö‡±á‡∞Ø‡∞°‡∞Ç**. ‡∞â‡∞¶‡∞æ‡∞π‡∞∞‡∞£‡∞ï‡±Å, ‡∞ï‡∞æ‡∞≤‡∞Æ‡±ç `A` ‡∞µ‡∞ø‡∞≤‡±Å‡∞µ 5 ‡∞ï‡∞Ç‡∞ü‡±á ‡∞é‡∞ï‡±ç‡∞ï‡±Å‡∞µ ‡∞â‡∞®‡±ç‡∞® ‡∞µ‡∞∞‡±Å‡∞∏‡∞≤‡∞®‡±Å ‡∞Æ‡∞æ‡∞§‡±ç‡∞∞‡∞Æ‡±á ‡∞â‡∞Ç‡∞ö‡∞æ‡∞≤‡∞Ç‡∞ü‡±á, ‡∞Æ‡±á‡∞Æ‡±Å `df[df['A']>5]` ‡∞Ö‡∞®‡∞ø ‡∞∞‡∞æ‡∞Ø‡∞µ‡∞ö‡±ç‡∞ö‡±Å.

> **‡∞ó‡∞Æ‡∞®‡∞ø‡∞ï**: ‡∞´‡∞ø‡∞≤‡±ç‡∞ü‡∞∞‡∞ø‡∞Ç‡∞ó‡±ç ‡∞é‡∞≤‡∞æ ‡∞™‡∞®‡∞ø‡∞ö‡±á‡∞∏‡±ç‡∞§‡±Å‡∞Ç‡∞¶‡∞Ç‡∞ü‡±á, `df['A']<5` ‡∞Ö‡∞®‡±á ‡∞µ‡±ç‡∞Ø‡∞ï‡±ç‡∞§‡±Ä‡∞ï‡∞∞‡∞£ ‡∞í‡∞ï ‡∞¨‡±Ç‡∞≤‡∞ø‡∞Ø‡∞®‡±ç ‡∞∏‡∞ø‡∞∞‡±Ä‡∞∏‡±ç‚Äå‡∞®‡±Å ‡∞á‡∞∏‡±ç‡∞§‡±Å‡∞Ç‡∞¶‡∞ø, ‡∞á‡∞¶‡∞ø ‡∞Ö‡∞∏‡∞≤‡±Å ‡∞∏‡∞ø‡∞∞‡±Ä‡∞∏‡±ç `df['A']` ‡∞≤‡±ã ‡∞™‡±ç‡∞∞‡∞§‡∞ø ‡∞Ö‡∞Ç‡∞∂‡∞Ç ‡∞ï‡±ã‡∞∏‡∞Ç ‡∞µ‡±ç‡∞Ø‡∞ï‡±ç‡∞§‡±Ä‡∞ï‡∞∞‡∞£ `True` ‡∞≤‡±á‡∞¶‡∞æ `False` ‡∞Ö‡∞®‡∞ø ‡∞∏‡±Ç‡∞ö‡∞ø‡∞∏‡±ç‡∞§‡±Å‡∞Ç‡∞¶‡∞ø. ‡∞¨‡±Ç‡∞≤‡∞ø‡∞Ø‡∞®‡±ç ‡∞∏‡∞ø‡∞∞‡±Ä‡∞∏‡±ç‚Äå‡∞®‡±Å ‡∞á‡∞Ç‡∞°‡±Ü‡∞ï‡±ç‡∞∏‡±ç‚Äå‡∞ó‡∞æ ‡∞â‡∞™‡∞Ø‡±ã‡∞ó‡∞ø‡∞∏‡±ç‡∞§‡±á, ‡∞Ö‡∞¶‡∞ø ‡∞°‡±á‡∞ü‡∞æ‡∞´‡±ç‡∞∞‡±á‡∞Æ‡±ç‚Äå‡∞≤‡±ã‡∞®‡∞ø ‡∞µ‡∞∞‡±Å‡∞∏‡∞≤ ‡∞â‡∞™‡∞∏‡∞Æ‡∞ø‡∞§‡∞ø‡∞®‡∞ø ‡∞á‡∞∏‡±ç‡∞§‡±Å‡∞Ç‡∞¶‡∞ø. ‡∞Ö‡∞Ç‡∞¶‡±Å‡∞µ‡∞≤‡±ç‡∞≤, ‡∞∏‡∞æ‡∞ß‡∞æ‡∞∞‡∞£ ‡∞™‡±à‡∞•‡∞æ‡∞®‡±ç ‡∞¨‡±Ç‡∞≤‡∞ø‡∞Ø‡∞®‡±ç ‡∞µ‡±ç‡∞Ø‡∞ï‡±ç‡∞§‡±Ä‡∞ï‡∞∞‡∞£‡∞®‡±Å ‡∞â‡∞™‡∞Ø‡±ã‡∞ó‡∞ø‡∞Ç‡∞ö‡∞°‡∞Ç ‡∞∏‡∞æ‡∞ß‡±ç‡∞Ø‡∞Ç ‡∞ï‡∞æ‡∞¶‡±Å, ‡∞â‡∞¶‡∞æ., `df[df['A']>5 and df['A']<7]` ‡∞§‡∞™‡±ç‡∞™‡±Å. ‡∞¨‡∞¶‡±Å‡∞≤‡±Å‡∞ó‡∞æ, ‡∞Æ‡±Ä‡∞∞‡±Å ‡∞¨‡±Ç‡∞≤‡∞ø‡∞Ø‡∞®‡±ç ‡∞∏‡∞ø‡∞∞‡±Ä‡∞∏‡±ç‚Äå‡∞≤‡∞™‡±à ‡∞™‡±ç‡∞∞‡∞§‡±ç‡∞Ø‡±á‡∞ï `&` ‡∞Ü‡∞™‡∞∞‡±á‡∞∑‡∞®‡±ç ‡∞â‡∞™‡∞Ø‡±ã‡∞ó‡∞ø‡∞Ç‡∞ö‡∞æ‡∞≤‡∞ø, ‡∞á‡∞≤‡∞æ ‡∞∞‡∞æ‡∞Ø‡∞æ‡∞≤‡∞ø `df[(df['A']>5) & (df['A']<7)]` (*‡∞¨‡±ç‡∞∞‡∞æ‡∞ï‡±Ü‡∞ü‡±ç‡∞≤‡±Å ‡∞á‡∞ï‡±ç‡∞ï‡∞° ‡∞Æ‡±Å‡∞ñ‡±ç‡∞Ø‡∞Æ‡±à‡∞®‡∞µ‡∞ø*).

**‡∞ï‡±ä‡∞§‡±ç‡∞§ ‡∞ó‡∞£‡∞®‡±Ä‡∞Ø‡∞Æ‡±à‡∞® ‡∞ï‡∞æ‡∞≤‡∞Æ‡±ç‚Äå‡∞≤‡∞®‡±Å ‡∞∏‡±É‡∞∑‡±ç‡∞ü‡∞ø‡∞Ç‡∞ö‡∞°‡∞Ç**. ‡∞Æ‡±á‡∞Æ‡±Å ‡∞∏‡±Å‡∞≤‡∞≠‡∞Ç‡∞ó‡∞æ ‡∞ï‡±ä‡∞§‡±ç‡∞§ ‡∞ó‡∞£‡∞®‡±Ä‡∞Ø‡∞Æ‡±à‡∞® ‡∞ï‡∞æ‡∞≤‡∞Æ‡±ç‚Äå‡∞≤‡∞®‡±Å ‡∞ï‡±ç‡∞∞‡∞ø‡∞Ç‡∞¶‡∞ø ‡∞µ‡∞Ç‡∞ü‡∞ø ‡∞∏‡±Ç‡∞ü‡∞ø‡∞ó‡∞æ ‡∞Ö‡∞∞‡±ç‡∞•‡∞Æ‡∞Ø‡±ç‡∞Ø‡±á ‡∞µ‡±ç‡∞Ø‡∞ï‡±ç‡∞§‡±Ä‡∞ï‡∞∞‡∞£ ‡∞â‡∞™‡∞Ø‡±ã‡∞ó‡∞ø‡∞Ç‡∞ö‡∞ø ‡∞∏‡±É‡∞∑‡±ç‡∞ü‡∞ø‡∞Ç‡∞ö‡∞µ‡∞ö‡±ç‡∞ö‡±Å:

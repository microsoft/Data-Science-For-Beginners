# ਡਾਟਾ ਸਾਇੰਸ ਦਾ ਪਰਿਚਯ

![ਡਾਟਾ ਦੀ ਕਾਰਵਾਈ](../../../translated_images/data.48e22bb7617d8d92188afbc4c48effb920ba79f5cebdc0652cd9f34bbbd90c18.pa.jpg)

–ï—Ç–æ –Ω—è–∫–æ–∏ –æ—Ç –Ω–∞–π-–≤–∞–∂–Ω–∏—Ç–µ –æ–ø–µ—Ä–∞—Ü–∏–∏, –∫–æ–∏—Ç–æ –º–æ–∂–µ–º –¥–∞ –∏–∑–≤—ä—Ä—à–∏–º –≤—ä—Ä—Ö—É DataFrames:

**–ò–∑–±–æ—Ä –Ω–∞ –∫–æ–ª–æ–Ω–∏**. –ú–æ–∂–µ–º –¥–∞ –∏–∑–±–µ—Ä–µ–º –æ—Ç–¥–µ–ª–Ω–∏ –∫–æ–ª–æ–Ω–∏, –∫–∞—Ç–æ –Ω–∞–ø–∏—à–µ–º `df['A']` - —Ç–∞–∑–∏ –æ–ø–µ—Ä–∞—Ü–∏—è –≤—Ä—ä—â–∞ Series. –ú–æ–∂–µ–º —Å—ä—â–æ –¥–∞ –∏–∑–±–µ—Ä–µ–º –ø–æ–¥–º–Ω–æ–∂–µ—Å—Ç–≤–æ –æ—Ç –∫–æ–ª–æ–Ω–∏ –≤ –¥—Ä—É–≥ DataFrame, –∫–∞—Ç–æ –Ω–∞–ø–∏—à–µ–º `df[['B','A']]` - —Ç–æ–≤–∞ –≤—Ä—ä—â–∞ –¥—Ä—É–≥ DataFrame.

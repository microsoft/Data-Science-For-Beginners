# スキルを活用しよう

## 手順

これまで、ミネソタ州の鳥類データセットを使って、鳥の数や個体密度に関する情報を調べてきました。これらの技術を別のデータセットで試して、実践的に活用してみましょう。[Kaggle](https://www.kaggle.com/)などからデータセットを取得してみてください。このデータセットについて物語を語るRスクリプトを作成し、ヒストグラムを使ってデータを説明することを忘れないでください。

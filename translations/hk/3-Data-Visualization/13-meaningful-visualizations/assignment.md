# 建立你自己的自訂視覺化

## 指引

使用此專案中的程式碼範例來建立一個社交網絡，模擬你自己的社交互動數據。你可以繪製你的社交媒體使用情況，或者製作一個家庭成員的圖表。創建一個有趣的網頁應用程式，展示一個獨特的社交網絡視覺化。

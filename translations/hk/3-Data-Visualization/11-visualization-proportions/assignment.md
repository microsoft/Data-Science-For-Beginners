# 在 Excel 中嘗試

## 指示

你知道嗎？你可以在 Excel 中建立甜甜圈圖、圓餅圖和鬆餅圖！使用你選擇的數據集，直接在 Excel 試算表中建立這三種圖表。

## 評分標準

| 優秀表現                                             | 合格表現                                      | 需要改進                                          |
| --------------------------------------------------- | --------------------------------------------- | ------------------------------------------------ |

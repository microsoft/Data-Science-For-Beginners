เดเดคเต เดจเดฟเดเตเดเตพเดเตเดเต เดธเตเดเตเดทเดฟเดเตเดเตเดฃเตเด เดกเดพเดฑเตเดฑเดฏเตเดเต เดตเดฒเดฟเดฏเตเดฐเต เดญเดพเดเด เดจเตเดเตเดเด เดเตเดฏเตเดฏเดพเด, เดชเตเดฐเดคเตเดฏเตเดเดฟเดเตเดเต เดเตเดฑเดฟเดฏ เดกเดพเดฑเตเดฑเดพเดธเตเดฑเตเดฑเตเดเดณเดฟเตฝ. เดจเดฟเดเตเดเตพเดเตเดเต เดเดเตเดเดฟเตฝ, เดจเตพ เดฎเตเดฒเตเดฏเดเตเดเตพ เดเดณเตเดณ เดเดฟเดฒ เดตเดฐเดฟเดเดณเต เดเตเดณเดเตเดเดณเต เดฎเดพเดคเตเดฐเด เดจเตเดเตเดเด เดเตเดฏเตเดฏเดฃเดฎเตเดเตเดเดฟเตฝ? `dropna`-เดฏเดฟเตฝ `how` เดเด `thresh` เดเด เดชเดพเดฐเดพเดฎเตเดฑเตเดฑเดฑเตเดเตพ เดเดชเดฏเตเดเดฟเดเตเดเต เดเดคเต เดจเดฟเตผเดฆเตเดฆเตเดถเดฟเดเตเดเดพเด.

- **เดจเตพ เดฎเตเดฒเตเดฏเดเตเดเตพ เดชเตเดฐเดฟเดชเตเดชเดฟเดเตเดเตฝ**: เดจเดฟเดเตเดเดณเตเดเต เดกเดพเดฑเตเดฑเดพเดธเตเดฑเตเดฑเดฟเดจเตเดฑเต เดเดเดฟเดธเตเดฅเดพเดจเดคเตเดคเดฟเตฝ, เดจเตพ เดฎเตเดฒเตเดฏเดเตเดเตพ เดจเตเดเตเดเด เดเตเดฏเตเดฏเตเดจเตเดจเดคเดฟเดจเต เดชเดเดฐเด เดธเดพเดงเตเดตเดพเดฏ เดฎเตเดฒเตเดฏเดเตเดเตพ เดเตเดฃเตเดเต เดชเตเดฐเดฟเดชเตเดชเดฟเดเตเดเตเดจเตเดจเดคเต เดเตเดเตเดคเตฝ เดฌเตเดฆเตเดงเดฟเดฎเตเดเตเดเดฟเดฒเตเดฒ. `isnull` เดเดชเดฏเตเดเดฟเดเตเดเต เดเดคเต เดเตเดฏเตเดฏเดพเด, เดชเดเตเดทเต เดเดคเต เดชเตเดฐเดฏเดพเดธเดเดฐเดฎเดพเดฏเดฟเดฐเดฟเดเตเดเตเด, เดชเตเดฐเดคเตเดฏเตเดเดฟเดเตเดเต เดชเตเดฐเดฟเดชเตเดชเดฟเดเตเดเตเดฃเตเด เดฎเตเดฒเตเดฏเดเตเดเตพ ๋ง์ผ๋ฉด. เดกเดพเดฑเตเดฑ เดธเดฏเตปเดธเดฟเตฝ เดเดคเต เดธเดพเดงเดพเดฐเดฃ เดชเตเดฐเดตเตผเดคเตเดคเดจเดฎเดพเดฃเตเดจเตเดจเต เดเตเดฃเตเดเต, pandas `fillna` เดจเตฝเดเตเดจเตเดจเต, เดเดคเต `Series` เดเดฒเตเดฒเตเดเตเดเดฟเตฝ `DataFrame`-เดจเตเดฑเต เดเดฐเต เดเตเดชเตเดชเดฟ เดจเตฝเดเตเดจเตเดจเต, เดจเดทเตเดเดชเตเดชเตเดเตเด เดฎเตเดฒเตเดฏเดเตเดเตพ เดจเดฟเดเตเดเตพ เดคเดฟเดฐเดเตเดเตเดเตเดเตเดเตเดจเตเดจ เดฎเตเดฒเตเดฏเดคเตเดคเตเดเต เดชเดเดฐเด เดตเดฏเตเดเตเดเตเดจเตเดจเต. เดชเตเดฐเดพเดฏเตเดเดฟเดเดฎเดพเดฏเดฟ เดเดคเต เดเดเตเดเดจเต เดชเตเดฐเดตเตผเดคเตเดคเดฟเดเตเดเตเดจเตเดจเตเดตเตเดจเตเดจเต เดเดพเดฃเดพเตป เดฎเดฑเตเดฑเตเดฐเต เดเดฆเดพเดนเดฐเดฃ `Series` เดธเตเดทเตเดเดฟเดเตเดเดพเด.

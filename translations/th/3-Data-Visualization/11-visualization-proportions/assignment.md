# ลองใช้ใน Excel

## คำแนะนำ

คุณรู้หรือไม่ว่าคุณสามารถสร้างกราฟโดนัท, กราฟพาย และกราฟวาฟเฟิลใน Excel ได้? ใช้ชุดข้อมูลที่คุณเลือก สร้างกราฟทั้งสามนี้ในสเปรดชีต Excel ของคุณ

### бҹЎ.бҹЎ бһҸбһҫ Azure Machine Learning бһҮбһ¶бһўбҹ’бһңбһё?

бһңбҹҒбһ‘бһ·бһҖбһ¶бһ–бһ–бһҖ Azure бһҳбһ¶бһ“бһ•бһӣбһ·бһҸбһ•бһӣбһ“бһ·бһ„бһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳбһ–бһ–бһҖбһҮбһ¶бһ„ бҹўбҹ бҹ  бһ“бһ·бһҷбһҳбһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҡбһ…бһ“бһ¶бһЎбһҫбһ„бһҠбһҫбһҳбҹ’бһ”бһёбһҮбһҪбһҷбһўбҹ’бһ“бһҖбһҡбҹҖбһ”бһ…бҹҶбһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһҗбҹ’бһҳбһёбҹ—бҹ”

бһўбҹ’бһ“бһҖбһңбһ·бһ‘бҹ’бһҷбһ¶бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ…бҹҶбһҺбһ¶бһҷбһ–бҹҒбһӣбһҷбҹүбһ¶бһ„бһ…бҹ’бһҡбһҫбһ“бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҹбһ·бһҖбҹ’бһҹбһ¶бһ“бһ·бһ„бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҮбһ¶бһҳбһ»бһ“ бһ“бһ·бһ„бһ–бҹ’бһҷбһ¶бһҷбһ¶бһҳбһ”бҹ’бһҡбһҫбһўбһ¶бһӣбҹӢбһҖбһ·бһҖбһҳбҹ’бһҷбһјбһҡ бһҖбһ¶бһҡбһ”бһҺбҹ’бһҠбһ»бҹҮбһ”бһҺбҹ’бһҠбһ¶бһӣбһҳбҹүбһјбһҠбҹӮбһӣ бһҸбһ¶бһҳбһҸбҹҶбһҺбһ¶бһҖбҹӢбһҖбһ¶бһӣбһҠбҹӮбһӣбһҳбһ¶бһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ– бһҠбһҫбһҳбҹ’бһ”бһёбһ•бһӣбһ·бһҸбһҳбҹүбһјбһҠбҹӮбһӣбһҳбһ¶бһ“бһ—бһ¶бһ–бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһңбҹ” бһҖбһ¶бһҡбһ„бһ¶бһҡбһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбһ…бҹҶбһҺбһ¶бһҷбһ–бҹҒбһӣбһҒбҹ’бһӣбһ¶бҹҶбһ„ бһ“бһ·бһ„бһҮбһ¶бһ„бһҸбһҳбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ§бһ”бһҖбһҡбһҺбҹҚбһӮбһҺбһ“бһ¶бһҠбҹӮбһӣбһҳбһ¶бһ“бһҗбҹ’бһӣбҹғбһҮбҹ’бһҡбһ¶бһ”бҹ”

[Azure ML](https://docs.microsoft.com/azure/machine-learning/overview-what-is-azure-machine-learning?WT.mc_id=academic-77958-bethanycheum&ocid=AID3041109) бһӮбһәбһҮбһ¶бһңбҹҒбһ‘бһ·бһҖбһ¶бһ–бһ–бһҖбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҹбҹ’бһҗбһ¶бһ”бһ“бһ¶ бһ“бһ·бһ„бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷ machine learning бһ“бҹ… Azureбҹ” бһңбһ¶бһҳбһ¶бһ“бһӣбһҖбҹ’бһҒбһҺбҹҲбһ–бһ·бһҹбҹҒбһҹбһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҮбһҪбһҷбһўбҹ’бһ“бһҖбһңбһ·бһ‘бҹ’бһҷбһ¶бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҡбҹҖбһ”бһ…бҹҶбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ бһ”бһҺбҹ’бһҠбһ»бҹҮбһ”бһҺбҹ’бһҠбһ¶бһӣбһҳбҹүбһјбһҠбҹӮбһӣ бһ”бҹ„бҹҮбһ–бһ»бһҳбҹ’бһ–бһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳбһ‘бһҹбҹ’бһҹбһ“бҹҚбһ‘бһ¶бһҷ бһ“бһ·бһ„бһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбҹ” бһ…бҹҶбһ”бһ„бһҮбһ¶бһ„бһ“бҹҒбҹҮ бһӮбһәбһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһҖбһҫбһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бһҠбҹ„бһҷбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһҖбһҳбҹ’бһҳбһ…бҹҶбһ–бҹ„бҹҮбһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ…бһ…бҹҶбһҺбһ¶бһҷбһ–бҹҒбһӣбһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бһ–бһ¶бһҖбҹӢбһ–бҹҗбһ“бҹ’бһ’бһ“бһ№бһ„бһҖбһ¶бһҡбһ”бһҺбҹ’бһҠбһ»бҹҮбһ”бһҺбҹ’бһҠбһ¶бһӣбһҳбҹүбһјбһҠбҹӮбһӣ бһ бһҫбһҷбһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһұбҹ’бһҷбһ”бҹ’бһҡбһҫбһ”бһүбҹ’бһҮбһёбһ’бһ“бһ’бһ¶бһ“бһӮбһҺбһ“бһ¶бһ–бһ–бһҖбһҠбҹӮбһӣбһўбһ¶бһ…бһ”бһ„бҹ’бһҖбһҫбһ“бһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ– бһҠбһҫбһҳбҹ’бһ”бһёбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ”бһҡбһ·бһҳбһ¶бһҺбһ’бҹҶбҹ— бһҒбһҺбҹҲбһҠбҹӮбһӣбһ”бһ„бҹӢбһҗбҹ’бһӣбҹғбһҸбҹӮбһ–бҹҒбһӣбһҠбҹӮбһӣбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ–бһ·бһҸбҹ”

Azure ML бһ•бҹ’бһҸбһӣбҹӢбһ§бһ”бһҖбһҡбһҺбҹҚбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“бҹҚ бһ“бһ·бһ„бһўбҹ’бһ“бһҖбһңбһ·бһ‘бҹ’бһҷбһ¶бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡ бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҲбһ»бһҸбһҖбһјбһҠ machine learning бһҡбһ”бһҹбҹӢбһ–бһҪбһҖбһӮбҹҒбҹ” бһҡбһ”бһҹбҹӢбһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбһҡбһҪбһҳбһҳбһ¶бһ“бҹ–

- **Azure Machine Learning Studio**: бһңбһ¶бһҮбһ¶бһӮбҹҒбһ бһ‘бҹҶбһ–бҹҗбһҡбһҳбһҪбһҷбһ“бҹ… Azure Machine Learning бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҮбһҳбҹ’бһҡбһҫбһҹбһӮбһјбһҠбһ‘бһ¶бһ” бһ“бһ·бһ„бһӮбҹ’бһҳбһ¶бһ“бһҖбһјбһҠбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бһҺбҹ’бһҠбһ»бҹҮбһ”бһҺбҹ’бһҠбһ¶бһӣбһҳбҹүбһјбһҠбҹӮбһӣ бһҖбһ¶бһҡбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ· бһҖбһ¶бһҡбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһҖбһҳбҹ’бһҳ бһҖбһ¶бһҡбһҸбһ¶бһҳбһҠбһ¶бһ“ бһ“бһ·бһ„бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ‘бҹ’бһҡбһ–бҹ’бһҷбһҹбһҳбҹ’бһ”бһҸбҹ’бһҸбһ·бҹ” Studio бһ”бһ¶бһ“бһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһҮбһ¶бһҳбһҪбһҷ Azure Machine Learning SDK бһҠбһҫбһҳбҹ’бһ”бһёбһҳбһ¶бһ“бһ”бһ‘бһ–бһ·бһҹбҹ„бһ’бһ“бҹҚбһҡбһӣбһјбһ“бҹ”
- **Jupyter Notebooks**: бһҸбһјбһ…бһңбһ·бһ—бһ¶бһӮбһ“бһ·бһ„бһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһҳбҹүбһјбһҠбҹӮбһӣ ML бһ”бһ¶бһ“бһҷбҹүбһ¶бһ„бһҡбһ бҹҗбһҹбҹ”
- **Azure Machine Learning Designer**: бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһұбҹ’бһҷбһ‘бһ¶бһүбһ“бһ·бһ„бһ‘бһҳбҹ’бһӣбһ¶бһҖбҹӢбһҳбҹҒбһ’бһјбһҠбһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһҖбһҫбһҸбһ”бһ‘бһ”бһ„бҹ’бһ бһ¶бһүбһ бһҫбһҷбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһҳбһҖбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһ“бһјбһңбһ•бҹ’бһӮбһ»бҹҶбһ”бһ“бҹ’бһ‘бһ¶бһҸбҹӢбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ”бһҡбһ·бһҷбһ¶бһҖбһ¶бһҹбһҖбһјбһҠбһ‘бһ¶бһ”бҹ”
- **UI бһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһҖбһҳбҹ’бһҳбһҡбҹҶбһ–бһ№бһ„бһ‘бһ»бһҖбһҳбҹүбһјбһҠбҹӮбһӣ (AutoML)**: бһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ…бһҳбһҪбһҷбһ…бҹҶбһ“бһҪбһ“бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһўбһ—бһ·бһңбһҢбҹ’бһҚбһҳбҹүбһјбһҠбҹӮбһӣ machine learning бһҠбһҫбһҳбҹ’бһ”бһёбһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһұбҹ’бһҷбһ”бһ„бҹ’бһҖбһҫбһҸбһҳбҹүбһјбһҠбҹӮбһӣ ML бһҳбһ¶бһ“бһҳбһ¶бһҸбҹ’бһҡбһҠбҹ’бһӢбһ¶бһ“бһҒбҹ’бһ–бһҹбҹӢ бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ– бһ“бһ·бһ„бһ•бһӣбһ·бһҸбһ—бһ¶бһ– бһҒбһҺбҹҲбһҖбһ¶бһҡбһҡбһҖбҹ’бһҹбһ¶бһӮбһ»бһҺбһ—бһ¶бһ–бһҳбҹүбһјбһҠбҹӮбһӣбҹ”

- **бһ•бҹ’бһ“бҹӮбһҖбһ”бһ“бҹ’бһҗбҹӮбһҳ machine learning бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ Visual Studio Code**: бһ•бҹ’бһҸбһӣбҹӢбһ”бһҡбһ·бһҷбһ¶бһҖбһ¶бһҹбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“бҹҚбһ–бҹҒбһүбһӣбҹҒбһүбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҹбҹ’бһҗбһ¶бһ”бһ“бһ¶ бһ“бһ·бһ„бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһӮбһҳбҹ’бһҡбҹ„бһ„ MLбҹ”
- **Command Line Interface machine learning**: бһ•бҹ’бһҸбһӣбҹӢбһ–бһ¶бһҖбҹ’бһҷбһ”бһүбҹ’бһҮбһ¶бһҠбһҫбһҳбҹ’бһ”бһёбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ’бһ“бһ’бһ¶бһ“ Azure ML бһҸбһ¶бһҳбһ”бһ“бҹ’бһ‘бһ¶бһҸбҹӢбһ–бһ¶бһҖбҹ’бһҷбһ”бһүбҹ’бһҮбһ¶бҹ”
- **бһҖбһ¶бһҡбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһҮбһ¶бһҳбһҪбһҷбһҹбҹҠбһ»бһҳбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ”бһҫбһҖбһ”бҹ’бһҡбһ—бһ–** бһҠбһјбһ…бһҮбһ¶ PyTorch, TensorFlow, Scikit-learn бһ“бһ·бһ„бһ”бһ“бҹ’бһҗбҹӮбһҳбһ‘бҹҖбһҸбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһ”бһҺбҹ’бһҠбһ»бҹҮбһ”бһҺбҹ’бһҠбһ¶бһӣ бһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ· бһ“бһ·бһ„бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҹбһҡбһ»бһ” machine learningбҹ”
- **MLflow**: бһңбһ¶бһӮбһәбһҮбһ¶бһ”бһҺбҹ’бһҺбһ¶бһӣбҹҗбһҷбһ”бһҫбһҖбһ”бҹ’бһҡбһ—бһ–бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҮбһёбһңбһ·бһҸбһҡбһ”бһҹбҹӢбһҖбһ¶бһҡбһ”бҹ’бһҡбһҳбһ·бһ‘бҹ’бһ’бһ”бһ‘бһ”бһ„бҹ’бһ бһ¶бһү machine learning бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ” **MLFlow Tracking** бһҮбһ¶вҖӢбһ”бҹ’бһҡвҖӢбһ—бһ¶бһӮвҖӢбһҳбһҪбһҷвҖӢбһ“бҹғ MLflow бһҠбҹӮбһӣбһҖбһҸбҹӢбһҸбҹ’бһҡбһ¶ бһ“бһ·бһ„бһҸбһ¶бһҳбһҠбһ¶бһ“бһҳбһ¶бһҸбҹ’бһҡбһҠбҹ’бһӢбһ¶бһ“бһҖбһ¶бһҡбһ”бһҺбҹ’бһҠбһ»бҹҮбһ”бһҺбҹ’бһҠбһ¶бһӣбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ бһ“бһ·бһ„бһўбһҸбҹ’бһҗбһ”бһ‘бһҳбҹүбһјбһҠбҹӮбһӣ бһ”бһҫбһҳбһ·бһ“бһӮбһ·бһҸбһҗбһ¶бһҮбҹҶбһ–бһјбһҖбһ”бһ‘бһ”бһ„бҹ’бһ бһ¶бһүбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ”бһҡбһ·бһҠбҹ’бһӢбһ¶бһ“бһҺбһ¶бҹ”

### бҹЎ.бҹў бһӮбһҳбҹ’бһҡбҹ„бһ„бһҹбһ“бҹ’бһ“бһ·бһҠбҹ’бһӢбһ¶бһ“бһ”бһҡбһ¶бһҮбҹҗбһҷбһ”бҹҒбҹҮбһҠбһјбһ„бҹ–

бһӮбҹ’бһҳбһ¶бһ“бһҖбһ¶бһҡбһҹбһ„бҹ’бһҹбҹҗбһҷбһҗбһ¶бһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһӮбһҳбҹ’бһҡбҹ„бһ„бһҮбһҪбһ”бһ”бҹ’бһҡбһ‘бҹҲбһҮбһ¶бһҗбҹ’бһҳбһёбһӮбһәбһҮбһ¶бһңбһ·бһ’бһёбһӣбҹ’бһўбһ”бҹҶбһ•бһ»бһҸбһҠбһҫбһҳбҹ’бһ”бһёбһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһҮбҹҶбһ“бһ¶бһүбһ“бһ·бһ„бһ…бҹҶбһҺбҹҒбҹҮбһҠбһ№бһ„бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ” бһҖбҹ’бһ“бһ»бһ„бһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮ бһҷбһҫбһ„бһ“бһ№бһ„бһҹбһ¶бһҖбһҹбһҪбһҡбһ–бһёбһ–бһёбһҡбһ•бҹ’бһӣбһјбһңбһҒбһ»бһҹбһӮбҹ’бһ“бһ¶бһ“бҹғбһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһӮбһҳбҹ’бһҡбҹ„бһ„бһңбһ·бһ‘бҹ’бһҷбһ¶бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҹбһ“бҹ’бһ“бһ·бһҠбҹ’бһӢбһ¶бһ“бһҖбһ¶бһҡбһ”бһҡбһ¶бһҮбҹҗбһҷбһ”бҹҒбҹҮбһҠбһјбһ„бһ“бҹ… Azure ML Studio бһҸбһ¶бһҳбһҡбһҷбҹҲ бһҖбһјбһҠбһ‘бһ¶бһ”/бһӮбҹ’бһҳбһ¶бһ“бһҖбһјбһҠ бһ“бһ·бһ„бһҸбһ¶бһҳбһҡбһҷбҹҲ Azure ML SDK бһҠбһјбһ…бһўбһҸбҹ’бһҗбһ¶бһ’бһ·бһ”бҹ’бһ”бһ¶бһҷбһҖбҹ’бһ“бһ»бһ„бһӮбҹҶбһ“бһјбһҡбһҹchematic бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбҹ–

### бҹЎ.бҹЈ бһҹбҹҶбһҺбһ»бҹҶбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ”бһҡбһ¶бһҮбҹҗбһҷбһ”бҹҒбҹҮбһҠбһјбһ„

бһҮбҹҶбһ„бһәбһ”бҹҒбҹҮбһҠбһјбһ„бһ“бһ·бһ„бһҹбһҡбһҹбҹғбһҲбһ¶бһҳ (CVDs) бһҮбһ¶бһўбһ„бҹ’бһӮбһ—бһ¶бһ–бһҳбһҪбһҷбһҠбҹӮбһӣбһ”бһҺбҹ’бһҸбһ¶бһӣбһІбҹ’бһҷбһўбҹ’бһ“бһҖбһҹбҹ’бһӣбһ¶бһ”бҹӢбһ…бҹ’бһҡбһҫбһ“бһӣбҹҒбһҒбҹЎбһӣбһҫбһ–бһ·бһ—бһ–бһӣбҹ„бһҖ бһҠбҹ„бһҷбһӮбһ·бһҸбһҮбһ¶бһҡбһҷбҹҲ бҹЈбҹЎ% бһ“бҹғбһҖбһ¶бһҡбһҹбҹ’бһӣбһ¶бһ”бҹӢбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһ“бҹ…бһӣбһҫбһ–бһ·бһ—бһ–бһӣбҹ„бһҖбҹ” бһҖбһҸбҹ’бһҸбһ¶бһҮбһ¶бһҹбҹ’бһҡбҹҒбһ…бһ”бһҡбһ·бһҷбһ¶бһҖбһ¶бһҹ бһ“бһ·бһ„бһўбһ¶бһҖбһ”бҹ’бһ”бһҖбһ·бһҡбһ·бһҷбһ¶ бһҠбһјбһ…бһҮбһ¶ бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ”бһ¶бһҡбһё бһўбһ¶бһ бһ¶бһҡбһҳбһ·бһ“бһҹбһҳбһҡбһҳбҹ’бһҷ бһ“бһ·бһ„бһ‘бһҳбҹ’бһ„бһ“бҹӢбһҖбҹ’бһҡбһ¶бһҹбҹӢ бһҳбһ¶бһ“бһҹбһҖбҹ’бһҸбһ¶бһ“бһ»бһ–бһӣбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бҹ’бһҡбһҫбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҳбҹүбһјбһҠбҹӮбһӣбһҖбҹҶбһҺбһҸбҹӢбһҸбһҳбҹ’бһӣбҹғбҹ” бһҖбһ¶бһҡбһўбһ¶бһ…бһ”бҹүбһ¶бһ“бҹӢбһҹбҹ’бһҳбһ¶бһ“бһўбһ¶бһ…бһ“бҹғбһҖбһ¶бһҡбһўбһ—бһ·бһңбһҢбҹ’бһҚбһҮбҹҶбһ„бһә CVD бһ“бһ№бһ„бһҳбһ¶бһ“бһ”бҹ’бһҡбһҷбҹ„бһҮбһ“бҹҚбһҠбһӣбҹӢбһҖбһ¶бһҡбһҖбһ¶бһҡбһ–бһ¶бһҡвҖӢбһҖбһ¶бһҡбһңбһ¶бһҷбһ”бҹ’бһҡбһ бһ¶бһҡбһ“бҹ…бһҳбһ“бһ»бһҹбҹ’бһҹбһҠбҹӮбһӣбһҳбһ¶бһ“бһ бһ¶бһ“бһ·бһ—бҹҗбһҷбһҒбҹ’бһ–бһҹбҹӢбҹ”

### бҹў.бҹЎ бһ”бһ„бҹ’бһҖбһҫбһҸбһҖбһ“бҹ’бһӣбҹӮбһ„бһ’бҹ’бһңбһҫбһҖбһ¶бһҡ Azure ML
бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһҺбҹ’бһҠбһ»бҹҮбһ”бһҺбҹ’бһҠбһ¶бһӣбһҳбҹүбһјбһҠбҹӮбһӣбһ“бҹ… Azure ML бһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһ”бһ„бҹ’бһҖбһҫбһҸбһҖбһ“бҹ’бһӣбҹӮбһ„бһ’бҹ’бһңбһҫбһҖбһ¶бһҡ Azure ML бһҮбһ¶бһҳбһ»бһ“бһҹбһ·бһ“бҹ” бһҖбһ“бҹ’бһӣбҹӮбһ„бһ’бҹ’бһңбһҫбһҖбһ¶бһҡ бһӮбһәбһ’бһ“бһ’бһ¶бһ“бһҳбһҪбһҷбһҖбҹҶбһ–бһјбһӣбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ Azure Machine Learning бһҠбҹӮбһӣбһ•бҹ’бһҸбһӣбҹӢбһҖбһ“бҹ’бһӣбҹӮбһ„бһҖбһҺбҹ’бһҸбһ¶бһӣбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһҮбһ¶бһҳбһҪбһҷбһўбһҸбҹ’бһҗбһ”бһ‘бһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ”бһ„бҹ’бһҖбһҫбһҸбһ“бҹ…бһ–бҹҒбһӣбһ”бҹ’бһҡбһҫ Azure Machine Learningбҹ” бһҖбһ“бҹ’бһӣбҹӮбһ„бһ’бҹ’бһңбһҫбһҖбһ¶бһҹбҹҶбһҒбһ¶бһ“бҹӢбҹ—бһҠбһјбһ…бһҮбһ¶ бһҡбһҖбҹ’бһҹбһ¶бһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһ”бһҺбҹ’бһҠбһ»бҹҮбһ”бһҺбҹ’бһҠбһ¶бһӣбһҡбһҪбһҳбһҳбһ¶бһ“бһҖбҹҶбһҺбһҸбҹӢбһ бҹҒбһҸбһ» бһҹбһ“бҹ’бһ‘бһҹбҹ’бһҹбһ“бҹҚ бһӣбһ‘бҹ’бһ’бһ•бһӣ бһ“бһ·бһ„бһҖбһ¶бһҡбһ”бһ¶бһүбҹӢбһ•бҹ’бһ‘бһ»бһҖбһҹбҹ’бһҖбҹ’бһҡбһёбһ”бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ” бһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһ“бҹҒбҹҮбһҠбһҫбһҳбҹ’бһ”бһёбһҖбҹҶбһҺбһҸбҹӢбһҗбһ¶бһҖбһ¶бһҡбһ”бһҺбҹ’бһҠбһ»бҹҮбһ”бһҺбҹ’бһҠбһ¶бһӣбһҺбһ¶бһ•бһӣбһ·бһҸбһҳбҹүбһјбһҠбҹӮбһӣбһӣбҹ’бһўбһ”бҹҶбһ•бһ»бһҸбҹ” [бһҹбһјбһҳбһҡбҹҖбһ“бһ”бһ“бҹ’бһҗбҹӮбһҳ](https://docs.microsoft.com/azure/machine-learning/concept-workspace?WT.mc_id=academic-77958-bethanycheum&ocid=AID3041109)

бһҳбһ¶бһ“бһҖбһ¶бһҡбһ•бҹ’бһҸбһӣбҹӢбһўбһ“бһ»бһҹбһ¶бһҹбһ“бҹҚбһІбҹ’бһҷбһ”бҹ’бһҡбһҫбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ»бһҖбһҡбһҖбһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“бһҠбҹӮбһӣбһҹбһҳбһҹбҹ’бһҡбһ”бһҮбһ¶бһҳбһҪбһҷбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ” бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ»бһҖбһҡбһҖбһҠбһјбһ…бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһӮбһ¶бҹҶбһ‘бҹ’бһҡбҹ–

- Microsoft Edge (Microsoft Edge бһҗбҹ’бһҳбһё бһҮбҹҶбһ“бһ¶бһ“бҹӢбһ…бһ»бһ„бһҖбҹ’бһҡбҹ„бһҷ бһҳбһ·бһ“бһҳбҹӮбһ“ Microsoft Edge бһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡ)
- Safari (бһҮбҹҶбһ“бһ¶бһ“бҹӢбһ…бһ»бһ„бһҖбҹ’бһҡбҹ„бһҷ бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ Mac бһҸбҹӮбһ”бҹүбһ»бһҺбҹ’бһҺбҹ„бҹҮ)

- Firefox (бһҮбҹҶбһ“бһ¶бһ“бҹӢбһ…бһ»бһ„бһҖбҹ’бһҡбҹ„бһҷ)

бһҠбһҫбһҳбҹ’бһ”бһёбһ”бҹ’бһҡбһҫ Azure Machine Learning бһҹбһјбһҳбһ”бһ„бҹ’бһҖбһҫбһҸбһҖбһ“бҹ’бһӣбҹӮбһ„бһ’бҹ’бһңбһҫбһҖбһ¶бһҡвҖӢбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҮбһ¶бһң Azure бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ” бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһҖбһ“бҹ’бһӣбҹӮбһ„бһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһ“бҹҒбҹҮбһҠбһҫбһҳбҹ’бһ”бһёбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ бһ’бһ“бһ’бһ¶бһ“бһӮбһҺбһ“бһ¶ бһҖбһјбһҠ бһҳбҹүбһјбһҠбҹӮбһӣ бһ“бһ·бһ„бһўбһҸбҹ’бһҗбһ”бһ‘бһ•бҹ’бһҹбҹҒбһ„бһ‘бҹҖбһҸбһҠбҹӮбһӣбһ‘бһ¶бһҖбҹӢбһ‘бһ„бһ“бһ№бһ„бһҖбһ¶бһҡбһ„бһ¶бһҡ machine learning бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ”

### бҹў.бҹў бһ’бһ“бһ’бһ¶бһ“бһӮбһҺбһ“бһ¶

бһ’бһ“бһ’бһ¶бһ“бһӮбһҺаёҷаёІ бһӮбһәбһҮбһ¶бһ’бһ“бһ’бһ¶бһ“бһ“бҹ…бһӣбһҫбһ–бһ–бһҖбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһ”бһҺбҹ’бһҠбһ»бҹҮбһ”бһҺбҹ’бһҠбһ¶бһӣбһҳбҹүбһјбһҠбҹӮбһӣ бһ“бһ·бһ„бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҹбҹ’бһҡбһ¶бһңбһҮбҹ’бһҡбһ¶бһңбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбҹ” бһҳбһ¶бһ“бһ”бһҪбһ“бһ”бҹ’бһҡбһ—бҹҒбһ‘бһ’бһ“бһ’бһ¶бһ“бһӮбһҺбһ“бһ¶бһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ”бһ„бҹ’бһҖбһҫбһҸбһ”бһ¶бһ“бҹ–
- **Compute Instances**: бһҹбҹ’бһҗбһ¶бһ“бһёбһҷбҹҚбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“бҹҚбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһңбһ·бһ‘бҹ’бһҷбһ¶бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһҠбһҫбһҳбҹ’бһ”бһёбһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһҮбһ¶бһҳбһҪбһҷбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ“бһ·бһ„бһҳбҹүбһјбһҠбҹӮбһӣбҹ” бһңбһ¶ бһҳбһ¶бһ“бһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒбһңбһёбһўбһ·бһҳбһҳбһҪбһҷ (VM) бһ бһҫбһҷбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһ”бҹ’бһҡбһҫбһҹбҹҖбһңбһ—бҹ…бһҖбҹҶбһҺбһҸбҹӢбһҸбҹ’бһҡбһ¶бҹ” бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһҳбһҖбһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ”бһ„бҹ’бһҡбҹҖбһ“бһҳбҹүбһјбһҠбҹӮбһӣбһҠбҹ„бһҷбһ бҹ…бһҖбҹ’бһҡбһ»бһҳбһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡбһ–бһёбһҹбҹҖбһңбһ—бҹ…бһҖбҹҶбһҺбһҸбҹӢбһҸбҹ’бһҡбһ¶бҹ”
- **Compute Clusters**: бһҖбҹ’бһҡбһ»бһҳбһўбҹ’бһ“бһҖбһңбһ·бһүбһңбҹҶбһ”бһҺбҹ’бһҸбһ¶бһүбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһңбһёбһўбһ·бһҳбһҠбҹӮбһӣбһўбһ¶бһ…бһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһҖбҹ’бһ“бһ»бһ„бһҹбҹҶбһҺбһ„бһҸбһ¶бһҳбһҸбһҳбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһ“бҹғбһҖбһјбһҠбһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бҹ” бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһңбһ¶вҖӢбһ–бҹҒбһӣбһ”бһ„бҹ’бһҡбҹҖбһ“бһҳбҹүбһјбһҠбҹӮбһӣбҹ” бһҖбҹ’бһҡбһ»бһҳбһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡбһ“бҹғбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһңбһёбһўбһ·бһҳбһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһ’бһ“бһ’бһ¶бһ“ GPU бһ¬ CPU бһҮбһ¶бһ–бһ·бһҹбҹҒбһҹбҹ”
- **Inference Clusters**: бһӮбҹ„бһӣбһҠбҹ…бһҖбһ¶бһҡбһ…бҹҒбһүбһ•бҹ’бһҹбһ¶бһҷбһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳбһ”бҹүбһ»бһ“бһ”бҹүбһ„бһҠбҹӮбһӣбһ”бҹ’бһҡбһҫбһҳбҹүбһјбһҠбҹӮбһӣбһҠбҹӮбһӣбһ”бһ¶бһ“бһ”бһ„бҹ’бһҡбҹҖбһ“бһҡбһҪбһ…бҹ”
- **Attached Compute**: бһ—бҹ’бһҮбһ¶бһ”бҹӢбһ‘бҹ…бһ“бһ№бһ„бһ’бһ“бһ’бһ¶бһ“бһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡбһўбһ¶бһ бҹҠбһәбһҡ Azure бһҳбһ¶бһ“бһҹбҹ’бһҡбһ¶бһ”бҹӢбһҠбһјбһ…бһҮбһ¶ бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһңбһёбһўбһ·бһҳ бһ¬бһҖбҹ’бһҡбһ»бһҳ Azure Databricks бҹ”

#### 2.2.1 бһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбһҮбһҳбҹ’бһҡбһҫбһҹбһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһңбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ’бһ“бһ’бһ¶бһ“бһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ

бһҳбһ¶бһ“бһҖбһҸбҹ’бһҸбһ¶бһҹбҹҶбһҒбһ¶бһ“бҹӢбҹ—бһҳбһҪбһҷбһ…бҹҶбһ“бһҪбһ“бһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһ–бһ·бһ…бһ¶бһҡбһҺбһ¶бһ“бҹҒбһӣбһ”бһ„бҹ’бһҖбһҫбһҸбһ’бһ“бһ’бһ¶бһ“бһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡ бһ бһҫбһҷбһҮбһҳбҹ’бһҡбһҫбһҹбһ‘бһ¶бҹҶбһ„бһ“бҹ„бҹҮбһўбһ¶бһ…бһҮбһ¶бһҖбһ¶бһҡбһҹбһҳбҹ’бһҡбҹҒбһ…бһ…бһ·бһҸбҹ’бһҸбһҹбҹҶбһҒбһ¶бһ“бҹӢбҹ”

**бһҸбһҫбһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҡбһ”бһҹбҹӢ CPU бһ¬ GPU ?**

CPU (Central Processing Unit) бһӮбһәбһҮбһ¶бһҹбҹҖбһӮбҹ’бһңбһёбһўбҹҒбһЎбһ·бһ…бһҸбҹ’бһҡбһјбһ“бһ·бһҖбһҠбҹӮбһӣбһўбһ“бһ»бһңбһҸбҹ’бһҸбһ“бҹҚбһҖбһ¶бһҡбһҺбҹӮбһ“бһ¶бҹҶбһҠбҹӮбһӣбһҳбһ¶бһ“бһҖбҹ’бһ“бһ»бһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡбҹ” GPU (Graphics Processing Unit) бһӮбһәбһҮбһ¶бһҹбҹҖбһӮбҹ’бһңбһёбһўбҹҒбһЎбһ·бһ…бһҸбҹ’бһҡбһјбһ“бһ·бһ…бһҮбһ¶бһҖбҹӢбһӣбһ¶бһҖбҹӢбһҳбһҪбһҷбһҠбҹӮбһӣбһўбһ¶бһ…бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҠбҹӮбһӣбһ‘бһ¶бһҖбҹӢбһ‘бһ„бһ“бһ№бһ„бһҖбҹ’бһҡбһ¶бһ бҹ’бһңбһ·бһҖбһ”бһ¶бһ“бһҷбҹүбһ¶бһ„бһӣбһҝбһ“бһҺбһ¶бһҹбҹӢбҹ”

бһ—бһ¶бһ–бһҒбһ»бһҹбһӮбҹ’бһ“бһ¶бһ…бһҳбҹ’бһ”бһ„бһҡбһңбһ¶бһ„бһҹбҹ’бһҗбһ¶бһ”бһҸбҹ’бһҷбһҖбһҳбҹ’бһҳ CPU бһ“бһ·бһ„ GPU бһӮбһә CPU бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҡбһ…бһ“бһ¶бһЎбһҫбһ„бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ…бһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бһҷбҹүбһ¶бһ„бһҡбһ бҹҗбһҹ (бһңбһ¶бһҹбҹӢбһҠбҹ„бһҷбһӣбҹ’бһ”бһҝбһ“бһҳбҹүбҹ„бһ„ CPU) бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһҳбһ¶бһ“бһҖбһҳбҹ’бһҡбһ·бһҸбһҖбҹҶбһҺбһҸбҹӢбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҡбһҸбҹӢбһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ…бһ…бҹ’бһҡбһҫбһ“бһ“бһ¶бһҖбҹӢбһҖбҹ’бһ“бһ»бһ„бһ–бҹҒбһӣбһҸбҹӮбһҳбһҪбһҷбҹ” GPU бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡбһҮбһ¶бһӮбһјбһҡбһҠбҹ„бһҷбһҹбһ¶бһҡбһ…бҹ’бһҡбһҫбһ“ бһ бһҫбһҷбһ бҹҒбһҸбһ»бһ“бҹҒбҹҮбһңбһ¶бһўбһ¶бһ…бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ„бһ¶бһҡбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҡбҹҖбһ“бһҮбҹ’бһҡбҹ…бһ”бһ¶бһ“бһҖбһ¶бһ“бҹӢбһҸбҹӮбһӣбҹ’бһўбҹ”

**бһ‘бҹҶбһ бҹҶбһҖбҹ’бһҡбһ»бһҳ**

бһҖбҹ’бһҡбһ»бһҳбһ’бҹҶбһ’бҹҒбһ„бһҳбһ¶бһ“бһҸбһҳбҹ’бһӣбҹғбһҗбҹ’бһӣбҹғбһҮбһ¶бһ„ бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһҳбһ¶бһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бһ”бҹҶбһӣбҹӮбһ„бһӣбҹ’бһўбһҮбһ¶бһ„бҹ” бһ бҹҒбһҸбһ»бһҠбһјбһ…бҹ’бһ“бҹҒбҹҮ бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһўбҹ’бһ“бһҖбһҳбһ¶бһ“бһ–бҹҒбһӣбһңбҹҒбһӣбһ¶бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһҳбһ·бһ“бһҳбһ¶бһ“бһӣбһ»бһҷбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ¶бһ“бҹӢ бһўбҹ’бһ“бһҖбһӮбһҪбһҡбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһҮбһ¶бһҳбһҪбһҷбһҖбҹ’бһҡбһ»бһҳбһҸбһјбһ…бҹ” бһ•бҹ’бһ‘бһ»бһҷбһЎбһҫбһҷ бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһўбҹ’бһ“бһҖбһҳбһ¶бһ“бһӣбһ»бһҷбһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһҳбһ·бһ“бһҳбһ¶бһ“бһ–бҹҒбһӣбһңбҹҒбһӣбһ¶бһ…бҹ’бһҡбһҫбһ“ бһўбҹ’бһ“бһҖбһӮбһҪбһҡбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһҮбһ¶бһҳбһҪбһҷбһҖбҹ’бһҡбһ»бһҳбһ’бҹҶбһҮбһ¶бһ„бҹ”

**бһ‘бҹҶбһ бҹҶ VM**

бһҷбҹ„бһ„бһ‘бҹ…бһӣбһҫбһ…бҹҶбһҺбһ»бһ…бһ–бҹ’бһҡбҹҶбһҠбҹӮбһ“бһ–бҹҒбһӣбһңбҹҒбһӣбһ¶бһ“бһ·бһ„бһҗбһңбһ·бһҖбһ¶бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ”бҹҶбһӣбҹӮбһ„бһ‘бҹҶбһ бҹҶ RAM, бһҢбһёбһҹ, бһ…бҹҶбһ“бһҪбһ“бһҖбҹҗбһҡбһ“бһ·бһ„бһӣбҹ’бһ”бһҝбһ“бһҳбҹүбҹ„бһ„бҹ” бһҖбһ¶бһҡбһ–бһ„бҹ’бһҡбһёбһҖбһ”бҹүбһ¶бһҡбҹүбһ¶бһҳбҹӮбһҸбҹ’бһҡвҖӢбһ‘бһ¶бҹҶбһ„бһ“бҹ„бҹҮбһ“бһ№бһ„бһҳбһ¶бһ“бһҗбҹ’бһӣбҹғбһҠбһјбһ…бһӮбҹ’бһ“бһ¶ бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһ“бһ№бһ„бһ”бһ„бҹ’бһҖбһҫбһҸбһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бһҖбһ¶бһ“бҹӢбһҸбҹӮбһ”бҹ’бһҡбһҹбһҫбһҡбҹ”

**бһҹбҹ’бһҗбһ¶бһ“бһёбһҷбҹҚбһ•бҹ’бһҸбҹ„бһҸбһ”бҹҶбһҡбһҫ бһ¬ бһҹбҹ’бһҗбһ¶бһ“бһёбһҷбҹҚбһўбһ¶бһ‘бһ·бһ—бһ¶бһ–бһ‘бһ¶бһ” ?**

бһҹбҹ’бһҗбһ¶бһ“бһёбһҷбҹҚбһўбһ¶бһ‘бһ·бһ—бһ¶бһ–бһ‘бһ¶бһ”бһҳбһ¶бһ“бһ“бҹҗбһҷбһҗбһ¶бһңбһ¶бһўбһ¶бһ…бһҸбҹ’бһҡбһјбһңбһ•бҹ’бһўбһ¶бһҖбһ”бһ¶бһ“бҹ– бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“ Microsoft Azure бһўбһ¶бһ…бһҷбһҖбһ’бһ“бһ’бһ¶бһ“бһ‘бһ¶бҹҶбһ„бһ“бҹ„бҹҮбһ“бһ·бһ„бһ•бҹ’бһҸбһӣбҹӢбһҮбһјбһ“бһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ…бһ•бҹ’бһҹбҹҒбһ„бһ‘бҹҖбһҸ бһҠбһјбһ…бҹ’бһ“бҹҒбҹҮбһ’бҹ’бһңбһҫбһІбҹ’бһҷбһҖбһ¶бһҡбһ„бһ¶бһҡбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ•бҹ’бһўбһ¶бһҖбҹ” бһҹбҹ’бһҗбһ¶бһ“бһёбһҷбҹҚбһ•бҹ’бһҸбҹ„бһҸбһ”бҹҶбһҡбһҫ бһ¬бһҳбһ·бһ“бһўбһ¶бһ…бһ•бҹ’бһўбһ¶бһҖбһ”бһ¶бһ“ бһҳбһ¶бһ“бһ“бҹҗбһҷбһҗбһ¶бһҖбһ¶бһҡбһ„бһ¶бһҡбһ“бһ№бһ„бһҳбһ·бһ“бһҸбҹ’бһҡбһјбһңбһ”бһүбҹ’бһ…бһ”бҹӢбһҠбҹ„бһҷбһӮбҹ’бһҳбһ¶бһ“бһҖбһ¶бһҡбһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһ–бһёбһўбҹ’бһ“бһҖбҹ” бһ“бҹҒбҹҮбһҮбһ¶бһҖбһ¶бһҡбһ–бһ·бһ…бһ¶бһҡбһҺбһ¶бһ•бҹ’бһҹбҹҒбһ„бһ‘бҹҖбһҸбһҡбһңбһ¶бһ„бһ–бҹҒбһӣбһңбҹҒбһӣбһ¶бһ“бһ·бһ„бһӣбһ»бһҷ бһ–бҹ’бһҡбҹ„бҹҮбһҹбҹ’бһҗбһ¶бһ“бһёбһҷбҹҚбһўбһ¶бһ‘бһ·бһ—бһ¶бһ–бһ‘бһ¶бһ”бһҳбһ¶бһ“бһҸбһҳбҹ’бһӣбҹғбһҗбҹ’бһӣбҹғбһҸбһ·бһ…бһҮбһ¶бһ„бһҹбҹ’бһҗбһ¶бһ“бһёбһҷбҹҚбһ•бҹ’бһҸбҹ„бһҸбһ”бҹҶбһҡбһҫбҹ”

#### 2.2.2 бһ”бһ„бҹ’бһҖбһҫбһҸ Compute Cluster

бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ [Azure ML workspace](https://ml.azure.com/) бһҠбҹӮбһӣбһҷбһҫбһ„бһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸбһ–бһёбһҳбһ»бһ“ бһ…бһјбһӣбһ‘бҹ…бһҖбһ¶бһ“бҹӢ compute бһ бһҫбһҷбһўбҹ’бһ“бһҖбһ“бһ№бһ„бһғбһҫбһүбһ’бһ“бһ’бһ¶бһ“ Compute бһ•бҹ’бһҹбҹҒбһ„бҹ—бһҠбҹӮбһӣбһҷбһҫбһ„бһ”бһ¶бһ“бһ–бһ·бһ—бһ¶бһҖбҹ’бһҹбһ¶бһҖбҹ’бһҡбҹ„бһҳбһ“бҹҒбҹҮ (бһҠбҹ„бһҷбһҳбһ¶бһ“ compute instances, compute clusters, inference clusters бһ“бһ·бһ„ attached compute)бҹ” бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһӮбһҳбҹ’бһҡбҹ„бһ„бһ“бҹҒбҹҮ бһҷбһҫбһ„бһ“бһ№бһ„бһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡ compute cluster бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бһ„бҹ’бһҡбҹҖбһ“бһҳбҹүбһјбһҠбҹӮбһӣ бһҖбҹ’бһ“бһ»бһ„ Studio бһ…бһ»бһ…бһӣбһҫбһҳбҹҒбһ“бһ»бһҷ "Compute" бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһҳбһҖбһҹбҹ’бһӣбһ¶бһ”бһ–бҹ’бһҡбһ·бһӣ "Compute cluster" бһ бһҫбһҷбһ…бһ»бһ…бһӣбһҫбһ”бҹҠбһјбһҸбһ»бһ„ "+ New" бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһҖбһҫбһҸ compute clusterбҹ”

### 2.4 бһ”бһ„бҹ’бһҡбҹҖбһ“бһҳбҹүбһјбһҠбҹӮбһӣбһҠбҹ„бһҷбһҖбһјбһҠбһҸбһ·бһ…/бһӮбҹ’бһҳбһ¶бһ“бһҖбһјбһҠбһ“бһ№бһ„ AutoML

бһҖбһ¶бһҡбһўбһ—бһ·бһңбһҢбҹ’бһҚбһҳбҹүбһјбһҠбҹӮбһӣбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҡбҹҖбһ“бһ”бҹӮбһ”бһ”бһ»бһҡбһ¶бһҺ бһҸбһҳбҹ’бһҡбһјбһңбһ’бһ“бһ’бһ¶бһ“бһ…бҹ’бһҡбһҫбһ“, бһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҮбҹҶбһ“бһ¶бһүбһ–бһ·бһҹбҹҒбһҹ бһ“бһ·бһ„бһ–бҹҒбһӣбһңбҹҒбһӣбһ¶бһ…бҹ’бһҡбһҫбһ“ бһҠбһҫбһҳбҹ’бһ”бһёбһ•бһӣбһ·бһҸбһ“бһ·бһ„бһ”бҹ’бһҡбҹҖбһ”бһ’бҹҖбһ”бһҳбҹүбһјбһҠбҹӮбһӣбһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бҹ”  
Automated machine learning (AutoML) бһӮбһәбһҮбһ¶бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһ”бҹҶбһ–бҹҒбһүбһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ…бһҠбҹӮбһӣбһ…бҹҶбһҺбһ¶бһҷбһ–бҹҒбһӣбһ…бҹ’бһҡбһҫбһ“ бһ“бһ·бһ„бһҳбһ·бһ“бһ…бһ”бҹӢбһ…бһ»бһ„бһ“бҹғбһҖбһ¶бһҡбһўбһ—бһ·бһңбһҢбҹ’бһҚбһҳбҹүбһјбһҠбҹӮбһӣбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҡбҹҖбһ“бҹ” бһңбһ¶бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһІбҹ’бһҷбһўбҹ’бһ“бһҖбһңбһ·бһ‘бҹ’бһҷбһ¶бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ бһўбҹ’бһ“бһҖбһңбһ·бһ—бһ¶бһӮ бһ“бһ·бһ„бһўбҹ’бһ“бһҖбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ”бҹ’бһӣбһ„бҹӢбһҖбһҹбһ¶бһ„бһҳбҹүбһјбһҠбҹӮбһӣ ML бһҠбҹ„бһҷбһҳбһ¶бһ“бһ‘бҹҶбһ бҹҶ, бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ– бһ“бһ·бһ„бһ•бһӣбһ·бһҸбһ—бһ¶бһ–бһҒбҹ’бһ–бһҹбҹӢ бһҒбһҺбҹҮбһ–бҹҒбһӣбһҡбһҖбҹ’бһҹбһ¶бһӮбһ»бһҺбһ—бһ¶бһ–бһҳбҹүбһјбһҠбҹӮбһӣбҹ” бһңбһ¶бһ’бҹ’бһңбһҫбһІбҹ’бһҷбһ–бҹҒбһӣбһңбҹҒбһӣбһ¶бһҠбһӣбҹӢбһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҳбҹүбһјбһҠбҹӮбһӣбһҖбҹ’бһ“бһ»бһ„бһўбһ¶бһҮбһёбһңбһҖбһҳбҹ’бһҳбһҸбһ·бһ…бһҮбһ¶бһ„бһ“бһ·бһ„бһҳбһ¶бһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бҹ” [бһҹбһјбһҳбһҹбҹ’бһңбҹӮбһ„бһҷбһӣбҹӢбһ”бһ“бҹ’бһҗбҹӮбһҳ](https://docs.microsoft.com/azure/machine-learning/concept-automated-ml?WT.mc_id=academic-77958-bethanycheum&ocid=AID3041109)

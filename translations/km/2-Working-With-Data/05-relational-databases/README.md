бһҖбһ¶бһҡбһҹбһҳбҹ’бһҡбһҪбһӣбһ—бһ¶бһ– бһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһ бҹӮбһӣбһҮбһ¶бһ”бһ¶бһ“бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҹбһ“бҹ’бһӣбһ№бһҖбһӮбһҺбһ“бһ¶ (spreadsheet) бһ“бҹ…бһўбһҸбһёбһҸбһҖбһ¶бһӣбһҠбһҫбһҳбҹ’бһ”бһёбһ•бҹ’бһ‘бһ»бһҖбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бҹ” бһўбҹ’бһ“бһҖбһҳбһ¶бһ“бһҮбһҪбһҡбһ“бһ·бһ„бһҖбһјбһЎбһ¶бҹҶ, бһҠбҹӮбһӣбһҮбһҪбһҡбһ“бҹ„бҹҮбһҳбһ¶бһ“бһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“ (бһ¬бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ), бһ бһҫбһҷбһҖбһјбһЎбһ¶бҹҶбһ–бһ·бһ–бһҺбҹҢбһ“бһ¶бһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“ (бһҠбҹӮбһӣбһҳбҹ’бһҸбһ„бһҳбҹ’бһҖбһ¶бһӣбһ бҹ…бһҗбһ¶metadata)бҹ” бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ‘бҹҶбһ“бһ¶бһҖбҹӢбһ‘бҹҶбһ“бһ„бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸбһЎбһҫбһ„бһӣбһҫбһӮбҹ’бһҡбһ№бҹҮбһҹбҹ’бһ“бһјбһӣбһ“бҹҒбҹҮбһ“бҹғбһҖбһјбһЎбһ¶бҹҶбһ“бһ·бһ„бһҮбһҪбһҡбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҸбһ¶бһҡбһ¶бһ„, бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһұбҹ’бһҷбһўбҹ’бһ“бһҖбһҳбһ¶бһ“бһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһ…бҹӮбһҖбһ…бһ¶бһҷбһҖбҹ’бһ“бһ»бһ„бһҸбһ¶бһҡбһ¶бһ„бһ…бҹ’бһҡбһҫбһ“бҹ” бһ“бҹҒбҹҮбһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһұбҹ’бһҷбһўбҹ’бһ“бһҖбһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһҮбһ¶бһҳбһҪбһҷбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҒбҹ’бһӣбһҪбһ“бһҹбҹ’бһҳбһ»бһӮбһҹбҹ’бһҳбһ¶бһүбһҮбһ¶бһ„бһ“бҹҒбҹҮ, бһҮбҹҖбһҹбһңбһ¶бһ„бһҖбһ¶бһҡбһ”бһүбҹ’бһ…бҹҒбһүбһ…бһҳбҹ’бһӣбһ„, бһ бһҫбһҷбһҳбһ¶бһ“бһ—бһ¶бһ–бһўбһ¶бһ…бһ”бһҸбҹӢбһ”бҹӮбһ“бһҖбҹ’бһ“бһ»бһ„бһңбһ·бһ’бһёбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбҹ” бһҷбһҫбһ„бһҳбһҖбһҹбҹ’бһңбҹӮбһ„бһҷбһӣбҹӢбһ–бһёбһӮбҹҶбһ“бһ·бһҸбһ“бҹғбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ‘бҹҶбһ“бһ¶бһҖбҹӢбһ‘бҹҶбһ“бһ„бҹ”

## бһңбһ·бһ’бһёбһӮбһ·бһҸбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһ–бһёбһҸбһ¶бһҡбһ¶бһ„

бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ‘бҹҶбһ“бһ¶бһҖбҹӢбһ‘бҹҶбһ“бһ„бһҳбһ¶бһ“бһ“бҹ…бһ…бһҳбҹ’бһ”бһ„бһҸбһ¶бһҡбһ¶бһ„бҹ” бһҠбһјбһ…бһҮбһ¶ spreadsheet, бһҸбһ¶бһҡбһ¶бһ„бһӮбһәбһҮбһ¶бһҖбһ¶бһҡбһ”бҹ’бһҡбһҳбһјбһӣбһ•бҹ’бһҸбһ»бҹҶбһҖбһјбһЎбһ¶бҹҶбһ“бһ·бһ„бһҮбһҪбһҡбҹ” бһҮбһҪбһҡбһҳбһ¶бһ“бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ¬бһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһҠбҹӮбһӣбһҷбһҫбһ„бһ…бһ„бҹӢбһ’бҹ’бһңбһҫбһҖбһ¶бһҡ бһҮбһ¶бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһҲбҹ’бһҳбҹ„бҹҮбһ‘бһёбһҖбҹ’бһҡбһ»бһ„бһ¬бһ”бһҡбһ·бһҳбһ¶бһҺбһ—бҹ’бһӣбҹҖбһ„бһ’бҹ’бһӣбһ¶бһҖбҹӢбҹ” бһҖбһјбһЎбһ¶бҹҶбһ–бһ·бһ–бһҺбҹҢбһ“бһ¶бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҠбҹӮбһӣбһ–бһҪбһҖбһӮбҹҒбһ•бҹ’бһ‘бһ»бһҖбҹ”

бһҷбһҫбһ„бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһҹбҹ’бһңбҹӮбһ„бһҷбһӣбҹӢбһҠбҹ„бһҷбһ”бһ„бҹ’бһҖбһҫбһҸбһҸбһ¶бһҡбһ¶бһ„бһҠбһҫбһҳбҹ’бһ”бһёбһ•бҹ’бһ‘бһ»бһҖбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһўбҹҶбһ–бһёбһ‘бһёбһҖбҹ’бһҡбһ»бһ„бҹ” бһҷбһҫбһ„бһўбһ¶бһ…бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһҮбһ¶бһҳбһҪбһҷбһҲбҹ’бһҳбҹ„бҹҮбһ“бһ·бһ„бһ”бҹ’бһҡбһ‘бҹҒбһҹбһҡбһ”бһҹбҹӢбһ–бһҪбһҖбһңбһ¶бҹ” бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ•бҹ’бһ‘бһ»бһҖбһңбһ¶бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҸбһ¶бһҡбһ¶бһ„бһҠбһјбһ…бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбҹ–

бһҹбһҳбҹ’бһӮбһ¶бһӣбҹӢбһҗбһ¶ бһҲбҹ’бһҳбҹ„бҹҮбһҖбһјбһЎбһ¶бҹҶ **бһ‘бһёбһҖбҹ’бһҡбһ»бһ„**, **бһ”бҹ’бһҡбһ‘бҹҒбһҹ** бһ“бһ·бһ„ **бһ”бҹ’бһҡбһҮбһ¶бһҮбһ“** бһ–бһ·бһ–бһҺбҹҢбһ“бһ¶бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҠбҹӮбһӣбһ”бһ¶бһ“бһ•бҹ’бһ‘бһ»бһҖ бһ бһҫбһҷбһҮбһҪбһҡбһҳбһҪбһҷбҹ—бһҳбһ¶бһ“бһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһўбҹҶбһ–бһёбһ‘бһёбһҖбҹ’бһҡбһ»бһ„бһҳбһҪбһҷбҹ”

## бһҳбһ·бһ“бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ¶бһ“бҹӢбһ“бҹғбһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҡбҹ’бһҸбһҸбһ¶бһҡбһ¶бһ„бһҸбҹӮбһҳбһҪбһҷ

бһ”бҹ’бһҡбһ бҹӮбһӣбһҮбһ¶бһҸбһ¶бһҡбһ¶бһ„бһҒбһ¶бһ„бһӣбһҫбһҳбһҫбһӣбһ‘бҹ…бһҹбҹ’бһӮбһ¶бһӣбҹӢбһҹбҹ’бһҡбһҠбҹҖбһ„бһӮбҹ’бһ“бһ¶ бҹ” бһҷбһҫбһ„бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһ”бһ“бҹ’бһҗбҹӮбһҳбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ”бһ“бҹ’бһҗбҹӮбһҳбһ‘бҹ…бһ‘бҹ…бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҖбҹҶбһ–бһ»бһ„бҹ– бһ—бҹ’бһӣбҹҖбһ„бһ”бҹ’бһҡбһ…бһ¶бҹҶбһҶбҹ’бһ“бһ¶бҹҶ (бһҮбһ¶ бһҳбһёбһӣбҹ’бһӣбһёбһҳбҹүбҹӮбһҸбҹ’бһҡ)бҹ” бһҷбһҫбһ„бһ“бһ№бһ„бһ•бҹ’бһҸбҹ„бһҸбһ‘бҹ…бһӣбһҫбһҶбҹ’бһ“бһ¶бҹҶ 2018, 2019 бһ“бһ·бһ„ 2020бҹ” бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһҷбһҫбһ„бһ”бһ“бҹ’бһҗбҹӮбһҳбһңбһ¶бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҸбһјбһӮбҹ’бһҷбһј бһңбһ¶бһўбһ¶бһ…бһҡбһҪбһҳбһ‘бһ¶бҹҶбһ„бһҠбһјбһ…бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбҹ–

бһҸбһҫбһўбҹ’бһ“бһҖбһ…бһ¶бһ”бҹӢбһўбһ¶бһҡбһҳбҹ’бһҳбһҺбҹҚбһўбҹ’бһңбһёбһҒбҹ’бһӣбҹҮбһ–бһёбһҸбһ¶бһҡбһ¶бһ„бһҡбһ”бһҹбҹӢбһҷбһҫбһ„? бһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһ бҹӮбһӣбһҮбһ¶бһҳбһҫбһӣбһғбһҫбһүбһҗбһ¶бһҷбһҫбһ„бһҖбҹҶбһ–бһ»бһ„бһ…бһҳбҹ’бһӣбһ„бһҲбҹ’бһҳбҹ„бҹҮбһ“бһ·бһ„бһ”бҹ’бһҡбһ‘бҹҒбһҹбһҡбһ”бһҹбҹӢбһ‘бһёбһҖбҹ’бһҡбһ»бһ„бһҳбҹ’бһҠбһ„бһҳбҹ’бһҖбһ¶бһӣбҹ” бһҗбҹ’бһӣбҹғбһҲбҹ’бһ“бһҪбһӣбһ•бҹ’бһ‘бһ»бһҖбһўбһ¶бһ…бһ”бһ„бҹ’бһҖбҹҒбһёбһҸбһҖбһ¶бһҡбһ…бҹҶбһҺбһ¶бһҷбһҒбҹ’бһӣбһ¶бҹҶбһ„ бһ бһҫбһҷбһңбһ¶бһҳбһ·бһ“бһ…бһ¶бҹҶбһ”бһ¶бһ…бҹӢбһҸбҹ’бһҡбһјбһңбһҳбһ¶бһ“бһ…бһҳбҹ’бһӣбһ„бһҳбһҪбһҷбһ…бҹҶбһ“бһҪбһ“бһ‘бҹҒбҹ” бһҮбһ¶бһ–бһ·бһҹбҹҒбһҹ бһҸбһјбһӮбҹ’бһҷбһјбһҳбһ¶бһ“бһҲбҹ’бһҳбҹ„бҹҮбһҸбҹӮбһҳбҹ’бһҠбһ„бһҠбҹӮбһӣбһҷбһҫбһ„бһ…бһ¶бһ”бҹӢбһўбһ¶бһҡбһҳбҹ’бһҳбһҺбҹҚбҹ”

бһӣбҹ’бһўбһ бһҫбһҷ бһҖбҹ’бһҡбҹ„бһҷбһҳбһҖбһҷбһҫбһ„бһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһҡбһҝбһ„бһ•бҹ’бһҹбҹҒбһ„бһ‘бҹҖбһҸбҹ” бһҷбһҫбһ„бһ”бһ“бҹ’бһҗбҹӮбһҳбһҖбһјбһЎбһ¶бҹҶбһҗбҹ’бһҳбһёбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһӮбҹ’бһҡбһ”бҹӢбһҶбҹ’бһ“бһ¶бҹҶбҹ—бҹ–

бһҳбһҖбһңбһ·бһүбһ‘бҹ…бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҡбһ”бһҹбҹӢбһҷбһҫбһ„ бһ бһҫбһҷбһҹбһҳбҹ’бһҡбҹҒбһ…бһ…бһ·бһҸбҹ’бһҸбһҗбһ¶бһўбҹ’бһ“бһҖбһ…бһ„бҹӢбһ”бҹҶбһ”бҹӮбһҖбһңбһ¶бһҷбҹүбһ¶бһ„бһҠбһјбһ…бһҳбҹ’бһҸбҹҒбһ…бҹ” бһҷбһҫбһ„бһҠбһ№бһ„бһҗбһ¶бһҷбһҫбһ„бһ…бһ„бҹӢбһ•бҹ’бһ‘бһ»бһҖбһҲбҹ’бһҳбҹ„бҹҮбһ“бһ·бһ„бһ”бҹ’бһҡбһ‘бҹҒбһҹбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ‘бһёбһҖбҹ’бһҡбһ»бһ„бһҡбһ”бһҹбҹӢбһҷбһҫбһ„, бһҠбһјбһ…бҹ’бһ“бҹҒбҹҮбһңбһ¶бһ“бһ№бһ„бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһӣбҹ’бһўбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҸбһ¶бһҡбһ¶бһ„бһҳбһҪбһҷбҹ”

> вң… бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһҹбһҳбҹ’бһӮбһ¶бһӣбҹӢбһҗбһ¶бһҷбһҫбһ„бһ”бҹ’бһҡбһҫбһ–бһ¶бһҖбҹ’бһҷ "id" бһ“бһ·бһ„ "primary key" бһҮбһ¶бһ”бҹ’бһҡбһ—бҹҒбһ‘бһ”бҹ’бһҡбҹҒбҹҮбһ”бҹ’бһҡбһ¶ бһ”бҹ’бһҡбһҮбһ»бҹҶбһҖбҹ’бһ“бһ»бһ„бһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮбҹ” бһӮбҹҶбһ“бһ·бһҸбһ“бҹ…бһ‘бһёбһ“бҹҒбҹҮбһҷбһҖбһ‘бҹ…бһ”бҹ’бһҡбһҫбһ”бһ¶бһ“бһҮбһ¶бһҳбһҪбһҷ DataFrames бһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ“бһ№бһ„бһҹбҹ’бһңбҹӮбһ„бһҷбһӣбҹӢбһ”бһ“бҹ’бһҸбһ·бһ…бһҖбҹ’бһҡбҹ„бһҷбҹ” DataFrames бһҳбһ·бһ“бһ”бҹ’бһҡбһҫбһ–бһ¶бһҖбҹ’бһҷ "primary key" бһ•бҹ’бһ‘бһ¶бһӣбҹӢбһ‘бҹҒ бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһўбҹ’бһ“бһҖбһ“бһ№бһ„бһҳбһҫбһӣбһғбһҫбһүбһҳбҹүбҹҒбһ…бһңбһ¶бһ”бҹ’бһҡбһ–бҹ’бһҡбһ№бһҸбҹ’бһҸбһ‘бҹ…бһҠбһјбһ…бһӮбҹ’бһ“бһ¶бҹ”

бһҮбһ¶бһҳбһҪбһҷбһҸбһ¶бһҡбһ¶бһ„бһ‘бһёбһҖбҹ’бһҡбһ»бһ„, бһҷбһҫбһ„бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһ•бҹ’бһ‘бһ»бһҖбһ—бҹ’бһӣбҹҖбһ„бҹ” бһ”бһҫбһңбһ¶бһҳбһ¶бһ“бһҖбһ¶бһҡбһ…бһҳбҹ’бһӣбһ„бһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһ–бҹҒбһүбһӣбҹҒбһүбһўбҹҶбһ–бһёбһ‘бһёбһҖбҹ’бһҡбһ»бһ„ бһҸбҹ’бһҡбһјбһңбһ”бҹ’бһҡбһҫ idбҹ” бһҷбһҫбһ„бһҸбҹ’бһҡбһјбһңбһ”бҹ’бһҡбһ¶бһҖбһҠбһҗбһ¶бһҸбһ¶бһҡбһ¶бһ„бһҗбҹ’бһҳбһёбһҖбҹҸбһҳбһ¶бһ“бһҖбһјбһЎбһ¶бҹҶ *id* бһ•бһ„бһҠбҹӮбһҡ бһ–бһёбһ–бҹ’бһҡбҹ„бҹҮбһҸбһ¶бһҡбһ¶бһ„бһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһӮбһҪбһҡбһҸбҹӮбһҳбһ¶бһ“ id бһ¬ primary keyбҹ”

## бһҖбһ¶бһҡбһ‘бһ¶бһүбһҷбһҖбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ

бһҠбҹ„бһҷбһҳбһ¶бһ“бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ”бҹҶбһ”бҹӮбһҖбһҮбһ¶ бҹў бһҸбһ¶бһҡбһ¶бһ„ бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ…бһ¶бһ”бҹӢбһўбһ¶бһҡбһҳбҹ’бһҳбһҺбҹҚбһҗбһ¶бһ“бһ№бһ„бһ‘бһ¶бһүбһҷбһҖбһңбһ¶бһҷбҹүбһ¶бһ„бһҠбһјбһ…бһҳбҹ’бһҸбҹҒбһ…бҹ” бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһҷбһҫбһ„бһ”бҹ’бһҡбһҫбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ‘бҹҶбһ“бһ¶бһҖбҹӢбһ‘бҹҶбһ“бһ„бһҠбһјбһ…бһҮбһ¶ MySQL, SQL Server бһ¬ Oracle, бһҷбһҫбһ„бһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһ—бһ¶бһҹбһ¶бһҳбһҪбһҷбһ бҹ…бһҗбһ¶ Structured Query Language бһ¬ SQLбҹ” SQL (бһҳбһҪбһҷбһ…бҹҶбһ“бһҪбһ“бһ бҹ…бһҗбһ¶ sequel) бһӮбһәбһҮбһ¶бһ—бһ¶бһҹбһ¶бһҳбһ¶бһҸбҹ’бһҡбһҷбҹ„бһ„бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ‘бһ¶бһүбһҷбһҖбһ“бһ·бһ„бһҖбҹӮбһ”бҹ’бһҡбҹӮбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҖбҹ’бһ“бһ»бһ„бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ‘бҹҶбһ“бһ¶бһҖбҹӢбһ‘бҹҶбһ“бһ„бҹ”

бһҳбһҖбһ‘бһ¶бһүбһҷбһҖбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ бһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһҖбһ¶бһҡбһ”бһүбҹ’бһҮбһ¶ `SELECT`бҹ” бһҠбһҫбһҳбһӮбҹҒ, бһўбҹ’бһ“бһҖ **бһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹ** бһҖбһјбһЎбһ¶бҹҶбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ…бһ„бҹӢбһҳбһҫбһӣ **бһ–бһё** бһҸбһ¶бһҡбһ¶бһ„бһҠбҹӮбһӣбһңбһ¶бһ•бҹ’бһ‘бһ»бһҖбһҮбһ»бҹҶбһңбһ·бһүбҹ” бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһўбҹ’бһ“бһҖбһ…бһ„бҹӢбһ”бһ„бҹ’бһ бһ¶бһүбһҸбҹӮбһҲбҹ’бһҳбҹ„бҹҮбһ‘бһёбһҖбҹ’бһҡбһ»бһ„ бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһҠбһјбһ…бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбҹ–

```sql
SELECT city
FROM cities;

-- Output:
-- Tokyo
-- Atlanta
-- Auckland
```
  
`SELECT` бһӮбһәбһҮбһ¶бһҖбһ“бҹ’бһӣбҹӮбһ„бһўбҹ’бһ“бһҖбһҡбһ¶бһ”бҹӢбһ”бһүбҹ’бһҮбһёбһҖбһјбһЎбһ¶бҹҶ бһ бһҫбһҷ `FROM` бһӮбһәбһҮбһ¶бһҖбһ“бҹ’бһӣбҹӮбһ„бһ”бһүбҹ’бһҮбһёбһҸбһ¶бһҡбһ¶бһ„бҹ”

> [!NOTE]  
> бһҹбҹҶбһҺбһ»бҹҶбһ”бҹӮбһ”бһ”бһ‘ SQL бһҳбһ·бһ“бһ”бҹ’бһҡбһҖбһ¶бһ“бҹӢбһҒбһҺбҹ’бһҢбһ–бһ»бһҳбҹ’бһ–бһўбһҖбҹ’бһҹбһҡбһ‘бҹҒ бһҳбһ¶бһ“бһ“бҹҗбһҷбһҗбһ¶ `select` бһ“бһ·бһ„ `SELECT` бһҳбһ¶бһ“бһ“бҹҗбһҷбһҠбһјбһ…бһӮбҹ’бһ“бһ¶бҹ” бһ‘бҹ„бҹҮбһҷбҹүбһ¶бһ„бһҺбһ¶, бһ•бҹ’бһўбҹӮбһҖбһ‘бҹ…бһӣбһҫбһ”бҹ’бһҡбһ—бҹҒбһ‘бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ бһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһ бҹӮбһӣбһҮбһ¶бһўбһ¶бһ…бһҳбһ¶бһ“бһҖбһҡбһҺбһёбһҖбһјбһЎбһ¶бҹҶбһ“бһ·бһ„бһҸбһ¶бһҡбһ¶бһ„бһҳбһ¶бһ“бһҖбһҡбһҺбһёбһ–бһ»бһҳбҹ’бһ–бһўбһҖбҹ’бһҹбһҡбһҖбҹҶбһҺбһҸбҹӢбҹ” бһҠбһјбһ…бҹ’бһ“бҹҒбҹҮ бһҮбһ¶бһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбһӣбҹ’бһўбһ”бҹҶбһ•бһ»бһҸ бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһўбҹ’бһ“бһҖбһңбһ¶бһҷбһҖбһјбһҠбһӮбһҪбһҡбһҸбҹӮбһ”бҹ’бһҡбһҫбһҡбһ”бҹҖбһ”бһҡбҹҖбһ”бһ…бҹҶбһ’бҹҶбһӣбһҫбһ–бһ¶бһҖбҹ’бһҷбһӮбһ“бҹ’бһӣбһ№бҹҮSQL бҹ”

бһҹбҹҶбһҺбһҪбһҡбһҒбһ¶бһ„бһӣбһҫбһ“бһ№бһ„бһ”бһ„бҹ’бһ бһ¶бһүбһ‘бһёбһҖбҹ’бһҡбһ»бһ„бһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбҹ” бһҠбһјбһ…бҹ’бһ“бҹҒбҹҮбһӮбһ·бһҸбһҗбһ¶ бһҷбһҫбһ„бһҸбҹ’бһҡбһјбһңбһ”бһ„бҹ’бһ бһ¶бһүбһҸбҹӮбһ‘бһёбһҖбҹ’бһҡбһ»бһ„бһҠбҹӮбһӣбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ New Zealandбҹ” бһҷбһҫбһ„бһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҹбҹҶбһҡбһ¶бһ”бҹӢбһҸбһҳбҹ’бһҡбһ„бһҳбһҪбһҷбҹ” бһ–бһ¶бһҖбҹ’бһҷ SQL бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ“бҹҒбҹҮбһӮбһә `WHERE`, бһ¬ "бһ“бҹ…бһҖбһ“бҹ’бһӣбҹӮбһ„бһҠбҹӮбһӣбһўбҹ’бһңбһёбһҳбһҪбһҷбһ–бһ·бһҸбҹ”"

```sql
SELECT city
FROM cities
WHERE country = 'New Zealand';

-- Output:
-- Auckland
```
  
## бһҖбһ¶бһҡбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ

бһҡбһ бһјбһҸбһҳбһҖбһҠбһӣбҹӢбһ–бҹҒбһӣбһ“бҹҒбҹҮбһҷбһҫбһ„бһ”бһ¶бһ“бһ‘бһ¶бһүбһҷбһҖбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷвҖӢбһ–бһёбһҸбһ¶бһҡбһ¶бһ„бһҸбҹӮбһҳбһҪбһҷбҹ” бһҘбһЎбһјбһңбһ“бҹҒбҹҮбһҷбһҫбһ„бһ…бһ„бҹӢбһҠбһ„бҹ’бһ бһҫбһҷбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ–бһёбһ‘бһ¶бҹҶбһ„бһ–бһёбһҡ **cities** бһ“бһ·бһ„ **rainfall**бҹ” бһңбһ¶бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҠбҹ„бһҷ *joining* бһ–бһҪбһҖбһңбһ¶бһҡбһҪбһҳбһӮбҹ’бһ“бһ¶бҹ” бһўбҹ’бһ“бһҖбһҳбһ¶бһ“бһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһ…бҹҶбһҺбһ»бһ…бһ—бҹ’бһҮбһ¶бһ”бҹӢбһҡбһңбһ¶бһ„бһҸбһ¶бһҡбһ¶бһ„бһ–бһёбһҡбһ“бҹ…бһҖбһ“бҹ’бһӣбҹӮбһ„бһҺбһ¶бһҳбһҪбһҷ бһ бһҫбһҷбһ—бҹ’бһҮбһ¶бһ”бҹӢбһҸбһҳбҹ’бһӣбҹғбһ–бһёбһҖбһјбһЎбһ¶бҹҶбһҳбһҪбһҷбһ–бһёбһҸбһ¶бһҡбһ¶бһ„бһҳбһҪбһҷбһ‘бҹ…бһҖбһјбһЎбһ¶бҹҶбһ–бһёбһҸбһ¶бһҡбһ¶бһ„бһҳбһҪбһҷбһ‘бҹҖбһҸбҹ”

бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһӣбҹҶбһ бһјбһҡбһҷбһҫбһ„бһҷбһҪбһҡбһҷбһҖбһҖбһјбһЎбһ¶бҹҶ **city_id** бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ **rainfall** бһҮбһ¶бһҳбһҪбһҷбһҖбһјбһЎбһ¶бҹҶ **city_id** бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ **cities**бҹ” бһңбһ¶бһ“бһ№бһ„бһ—бҹ’бһҮбһ¶бһ”бҹӢбһҸбһҳбҹ’бһӣбҹғбһ—бҹ’бһӣбҹҖбһ„бһҮбһ¶бһҳбһҪбһҷбһ‘бһёбһҖбҹ’бһҡбһ»бһ„бһ–бһ¶бһҖбҹӢбһ–бҹҗбһ“бҹ’бһ’бһҡбһ”бһҹбҹӢбһңбһ¶бҹ” бһ”бҹ’бһҡбһ—бҹҒбһ‘ join бһҠбҹӮбһӣбһҷбһҫбһ„бһ“бһ№бһ„бһ’бҹ’бһңбһҫбһ бҹ…бһҗбһ¶ *inner* join, бһҳбһ¶бһ“бһ“бҹҗбһҷбһҗбһ¶бһ”бһҫбһҮбһҪбһҡбһҺбһ¶бһҳбһҪбһҷбһҳбһ·бһ“бһ”бҹүбҹҮбһӮбҹ’бһ“бһ¶бһ“бһ№бһ„бһўбҹ’бһңбһёбһҳбһҪбһҷбһ–бһёбһҸбһ¶бһҡбһ¶бһ„бһ•бҹ’бһҹбҹҒбһ„ бһңбһ¶бһ“бһ№бһ„бһҳбһ·бһ“бһ”бһ„бҹ’бһ бһ¶бһүбһЎбһҫбһ„бһ‘бҹҒбҹ” бһҖбҹ’бһ“бһ»бһ„бһҡбһҝбһ„бһҡбһ”бһҹбҹӢбһҷбһҫбһ„ бһҸбһҫбһҳбһҪбһҷбҹ—бһ‘бһёбһҖбҹ’бһҡбһ»бһ„бһҳбһ¶бһ“бһ—бҹ’бһӣбҹҖбһ„бһ‘бҹҒ, бһҠбһјбһ…бҹ’бһ“бҹҒбҹҮбһўбҹ’бһңбһёбҹ—бһӮбҹ’бһҡбһ”бҹӢбһҷбҹүбһ¶бһ„бһ“бһ№бһ„бһ”бһ„бҹ’бһ бһ¶бһүбҹ”

бһҷбһҫбһ„бһҹбһ„бҹӢбһҹбҹҶбһҺбһјбһҳбһ–бһҡбһ‘бһ¶бһүбһҷбһҖбһ—бҹ’бһӣбҹҖбһ„бһҡбһ”бһҹбҹӢбһҶбҹ’бһ“бһ¶бҹҶ 2019 бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ‘бһёбһҖбҹ’бһҡбһ»бһ„бһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбҹ”

бһҷбһҫбһ„бһ“бһ№бһ„бһ’бҹ’бһңбһҫбһңбһ¶бһҮбһ¶бһҮбҹҶбһ бһ¶бһ“бҹ—бҹ” бһҮбҹҶбһ бһ¶бһ“бһҠбҹҶбһ”бһјбһ„бһӮбһәбһ”бһүбҹ’бһ…бһјбһӣбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҡбһҪбһҳбһӮбҹ’бһ“бһ¶ бһҠбҹ„бһҷбһ”бһ„бҹ’бһ бһ¶бһүбһҖбһјбһЎбһ¶бҹҶбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ…бҹҶбһҺбһ»бһ…бһ—бҹ’бһҮбһ¶бһ”бҹӢ - **city_id** бһҠбһјбһ…бһҠбҹӮбһӣбһ”бһ¶бһ“бһ”бһ„бҹ’бһ бһ¶бһүбһҒбһ¶бһ„бһӣбһҫбҹ”

```sql
SELECT cities.city
    rainfall.amount
FROM cities
    INNER JOIN rainfall ON cities.city_id = rainfall.city_id
```
  
бһҷбһҫбһ„бһ”бһ¶бһ“бһ–бһҺбҹҢбһ”бһ„бҹ’бһ бһ¶бһүбһ–бһёбһҖбһјбһЎбһ¶бҹҶбһ‘бһ¶бҹҶбһ„бһ–бһёбһҡбһҠбҹӮбһӣбһҷбһҫбһ„бһ…бһ„бҹӢбһ”бһ¶бһ“ бһ бһҫбһҷбһҸбҹ’бһҡбһјбһңбһ…бһјбһӣбһҡбһҪбһҳбһҸбһ¶бһҡбһ¶бһ„бһҠбҹ„бһҷ **city_id**бҹ” бһҘбһЎбһјбһңбһ“бҹҒбҹҮбһўбһ¶бһ…бһ”бһ“бҹ’бһҗбҹӮбһҳбһ”бҹ’бһҡбһҷбҹ„бһӮ `WHERE` бһҠбһҫбһҳбҹ’бһ”бһёбһҸбһҳбҹ’бһҡбһ„бһўбҹҶбһ–бһёбһҶбҹ’бһ“бһ¶бҹҶ 2019 бһ”бҹүбһ»бһҺбҹ’бһҺбҹ„бҹҮбҹ”

```sql
SELECT cities.city
    rainfall.amount
FROM cities
    INNER JOIN rainfall ON cities.city_id = rainfall.city_id
WHERE rainfall.year = 2019

-- Output

-- city     | amount
-- -------- | ------
-- Tokyo    | 1874
-- Atlanta  | 1111
-- Auckland |  942
```
  
## бһҹбһҡбһ»бһ”бһҳбҹҒбһҡбҹҖбһ“

бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ‘бҹҶбһ“бһ¶бһҖбҹӢбһ‘бҹҶбһ“бһ„бһ•бҹ’бһўбҹӮбһҖбһӣбһҫбһҖбһ¶бһҡбһ”бҹҶбһ”бҹӮбһҖбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бһҸбһ¶бһҡбһ¶бһ„бһҠбҹӮбһӣбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһҳбһҖбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҷбһҖбһҳбһҖбһңбһ·бһүбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бһ„бҹ’бһ бһ¶бһүбһ“бһ·бһ„бһңбһ·бһ—бһ¶бһӮбҹ” бһңбһ¶бһ–бһ·бһ…бһ¶бһҡбһҺбһ¶бһҮбһ¶бһҖбһҳбҹ’бһҡбһ·бһҸбһҒбҹ’бһ–бһҹбҹӢбһ“бҹғбһ—бһ¶бһ–бһ”бһҸбҹӢбһ”бҹӮбһ“бһҠбһҫбһҳбҹ’бһ”бһёбһӮбһҺбһ“бһ¶бһ“бһ·бһ„бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ•бҹ’бһҹбҹҒбһ„бһ‘бҹҖбһҸбҹ” бһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһғбһҫбһүбһҳбҹҒбһҡбҹҖбһ“бһӮбҹ’бһҡбһ№бҹҮбһ“бҹғбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ‘бҹҶбһ“бһ¶бһҖбҹӢбһ‘бҹҶбһ“бһ„ бһ бһҫбһҷбһ”бһ¶бһ“бһҖбһ·бһ…бҹ’бһ…бһҖбһ¶бһҡбһ”бһүбҹ’бһ…бһјбһӣбһӮбҹ’бһ“бһ¶бһҡбһңбһ¶бһ„бһҸбһ¶бһҡбһ¶бһ„бһ–бһёбһҡбҹ”

## рҹҡҖ мұҢлҰ°м§Җ

бһҳбһ¶бһ“бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ‘бҹҶбһ“бһ¶бһҖбҹӢбһ‘бҹҶбһ“бһ„бһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бһҠбҹӮбһӣбһҳбһ¶бһ“бһҹбҹ’бһҡбһ¶бһ”бҹӢбһ“бҹ…бһӣбһҫбһўбҹҠбһёбһ“бһ’бһәбһҺбһ·бһҸбҹ” бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҹбҹ’бһңбҹӮбһ„бһҷбһӣбҹӢбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҠбҹ„бһҷбһ”бҹ’бһҡбһҫбһҮбҹҶбһ“бһ¶бһүбһҠбҹӮбһӣбһ”бһ¶бһ“бһҡбҹҖбһ“бһҒбһ¶бһ„бһӣбһҫбҹ”

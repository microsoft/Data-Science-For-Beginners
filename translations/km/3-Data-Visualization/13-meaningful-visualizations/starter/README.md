# бһӮбһҳбҹ’бһҡбҹ„бһ„бһҖбһ¶бһҡбһҸбһ¶бһҖбҹӢбһҸбҹӮбһ„бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ Dangerous Liaisons

бһҠбһҫбһҳбҹ’бһ”бһёбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳ бһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһ”бҹ’бһҡбһ¶бһҖбһҠбһҗбһ¶ бһўбҹ’бһ“бһҖбһҳбһ¶бһ“ NPM бһ“бһ·бһ„ Node бһҖбҹҶбһ–бһ»бһ„бһҡбһҸбҹӢбһӣбһҫбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ” бһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһҸбһҳбҹ’бһӣбһҫбһ„бһўбһ¶бһҹбҹ’бһҡбҹҗбһҷбһ—бһ¶бһ– (npm install) бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһҳбһҖбһҡбһҸбҹӢбһӮбһҳбҹ’бһҡбҹ„бһ„бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһ“бҹ’бһӣбҹӮбһ„бһҹбҹ’бһҡбһ»бһҖ (npm run serve):

## бһҖбһ¶бһҡбһҸбһҳбҹ’бһӣбһҫбһ„бһӮбһҳбҹ’бһҡбҹ„бһ„
```
npm install
```

### бһҖбһјбһҠбһ”бһүбҹ’бһ…бһјбһӣ бһ“бһ·бһ„бһ”бһүбҹ’бһ…бһјбһӣбһЎбһҫбһ„бһңбһ·бһүбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһўбһ—бһ·бһңбһҢбҹ’бһҚ
```
npm run serve
```

### бһҖбһјбһҠбһ”бһүбҹ’бһ…бһјбһӣ бһ“бһ·бһ„бһ”бһ„бҹ’бһҡбһҪбһҳбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһ•бһӣбһ·бһҸ
```
npm run build
```

### бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҖбһјбһҠ бһ“бһ·бһ„бһҖбҹӮбһҸбһҳбҹ’бһҡбһјбһңбһҜбһҖбһҹбһ¶бһҡ
```
npm run lint
```

### бһҖбҹҶбһҺбһҸбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҒбҹ’бһӣбһҪбһ“

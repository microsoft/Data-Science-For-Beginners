# Dangerous Liaisons бһӮбһҳбҹ’бһҡбҹ„бһ„бһҖбһ¶бһҡбһҳбһҫбһӣбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ

бһҠбһҫбһҳбҹ’бһ”бһёбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳ бһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһҸбҹӮбһ’бһ¶бһ“бһ¶бһҗбһ¶ бһўбҹ’бһ“бһҖбһҳбһ¶бһ“ NPM бһ“бһ·бһ„ Node бһҡбһҸбҹӢбһ“бҹ…бһӣбһҫбһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ” бһҸбһҳбҹ’бһӣбһҫбһ„бһҖбһ¶бһҡбһӮбһ¶бҹҶбһ‘бҹ’бһҡ (npm install) бһ бһҫбһҷбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһҳбһҖбһҡбһҸбҹӢбһӮбһҳбҹ’бһҡбҹ„бһ„бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҸбҹҶбһ”бһ“бҹӢбһҖбҹ’бһ“бһ»бһ„ (npm run serve):

## бһҖбһ¶бһҡбһҸбҹҶбһЎбһҫбһ„бһӮбһҳбҹ’бһҡбҹ„бһ„
```
npm install
```

### бһ”бһҖбһҹбҹ’бһҡбһ¶бһҷ бһ“бһ·бһ„бһ”бһүбҹ’бһ…бһјбһӣбһЎбһҫбһ„бһңбһ·бһүбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“бҹҚ
```
npm run serve
```

### бһ”бһҖбһҹбҹ’бһҡбһ¶бһҷ бһ“бһ·бһ„бһ”бһ„бҹ’бһҡбһҪбһҳбһ…бҹҶбһҺбһ»бһ…бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ•бһӣбһ·бһҸбһҖбһҳбҹ’бһҳ
```
npm run build
```

### бһ–бһ·бһ“бһ·бһҸбҹ’бһҷ бһ“бһ·бһ„бһҖбҹӮбһҹбһҳбҹ’бһҡбһҪбһӣбһҜбһҖбһҹбһ¶бһҡ
```
npm run lint
```

### бһҖбҹҶбһҺбһҸбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҒбҹ’бһӣбһҪбһ“
бһҳбһҫбһӣ [бһҖбһ¶бһҡбһ”бһүбҹ’бһҮбһ¶бһҖбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’](https://cli.vuejs.org/config/)гҖӮ

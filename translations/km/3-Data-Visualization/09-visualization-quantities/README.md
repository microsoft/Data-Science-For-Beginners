## бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҳбһҫбһӣбһ‘бҹҶбһ бҹҶбһ“бҹғбһҹбҹ’бһӣбһ¶бһ”бһҮбһ¶бһҳбһҪбһҷ Matplotlib

бһ”бһҺбҹ’бһҺбһ¶бһӣбҹҗбһҷбһӣбҹ’бһўбһҳбһҪбһҷбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бһ„бҹ’бһҖбһҫбһҸбһҖбҹ’бһҡаёІаёҹбһҹбһ¶бһҳбһүбҹ’бһү бһ“бһ·бһ„бһҹбҹ’бһҳбһ»бһӮбһҹбҹ’бһҳбһ¶бһүбһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бһӮбһә [Matplotlib](https://matplotlib.org/stable/index.html)бҹ” бһҮбһ¶бһҡбһҪбһҳ бһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһ“бҹғбһҖбһ¶бһҡбһӮбһјбһҡбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҠбҹ„бһҷбһ”бҹ’бһҡбһҫбһ”бһҺбҹ’бһҺбһ¶бһӣбҹҗбһҷбһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбһҡбһҪбһҳбһҳбһ¶бһ“ бһҖбһ¶бһҡбһҡбһҖбһғбһҫбһүбһ•бҹ’бһ“бҹӮбһҖбһ“бҹғбһўбһ„бҹӢбһҸбҹ’бһҡбһјбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҠбһ¶бһҸбһ¶бһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ…бһ„бҹӢбһ•бҹ’бһҸбҹ„бһҸ, бһҖбһ¶бһҡбһ”бҹ’бһҡбҹӮбһҖбҹ’бһӣбһ¶бһҷбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҠбҹӮбһӣбһ…бһ¶бҹҶбһ”бһ¶бһ…бҹӢ, бһҖбһ¶бһҡбһ•бҹ’бһҠбһӣбҹӢбһҸбһҳбҹ’бһӣбҹғ axis x бһ“бһ·бһ„ y, бһҖбһ¶бһҡбһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбһ”бҹ’бһҡбһ—бҹҒбһ‘бһҖбҹ’бһҡбһ¶бһ•бһҸбҹ’бһҡбһёбһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһ”бһ„бҹ’бһ бһ¶бһү бһ бһҫбһҷбһ”бһ„бҹ’бһ бһ¶бһүбһ”бҹ’бһҡбһ¶бһ”бҹӢбҹ” Matplotlib бһ•бҹ’бһҸбһӣбҹӢбһҮбһјбһ“бһ“бһјбһңбһҖбһ¶бһҡбһҳбһҫбһӣбһғбһҫбһүбһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бһ”бҹҶбһ•бһ»бһҸ бһ”бҹүбһ»бһ“бҹ’бһҸбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһҳбһҖ бһҸбҹ„бҹҮбһ•бҹ’бһҠбҹ„бһҸбһ‘бҹ…бһӣбһҫбһҡбһјбһ”бһҳбһ“бҹ’бһҸбһҠбҹӮбһӣбһҹбһҳбһҹбҹ’бһҡбһ”бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһҳбһҫбһӣбһғбһҫбһүбһ…бҹҶбһ“бһҪбһ“бҹ– бһӮбҹҶбһ“бһјбһҡбһҮбһҪбһҡбһҲбһҡ бһҡбһјбһ” Scatter бһ“бһ·бһ„бһҡбһјбһ”бһ”бһ¶бһҡбҹ”

> вң… бһ”бҹ’бһҡбһҫбһҖбҹ’бһҡбһ¶бһ•бһҸбҹ’бһҡбһёбһӣбҹ’бһўбһ”бҹҶбһ•бһ»бһҸбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ бһ“бһ·бһ„бһҡбһҝбһ„бһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ…бһ„бҹӢбһ”бҹ’бһҡбһ¶бһ”бҹӢбҹ”
> - бһҠбһҫбһҳбҹ’бһ”бһёбһңбһ·бһ—бһ¶бһӮбһ“бһ·бһ“бҹ’бһ“бһ¶бһҖбһ¶бһҡбһҮбһ¶бһҳбһҪбһҪбһҷбһ”бһ“бҹ’бһ‘бһ¶бһҸбҹӢбһ–бҹҒбһӣбһңбҹҒбһӣбһ¶: бһҮбһҪбһҡбһҲбһҡ
> - бһҠбһҫбһҳбҹ’бһ”бһёбһ”бҹ’бһҡбҹҖбһ”бһ’бҹҖбһ”бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ: бһ”бһ¶бһҡ, бһҮбһҪбһҡбһҲбһҡ, бһҒбҹ’бһ“бһ¶бһҸбһҳбһјбһӣ, бһҡбһјбһ” scatterplot
> - бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһ бһ¶бһүбһ–бһёбһҡбһ”бҹҖбһ”бһҠбҹӮбһӣбһ•бҹ’бһ“бҹӮбһҖбһ•бҹ’бһҹбҹҒбһ„бһ‘бһ¶бһҖбҹӢбһ‘бһ„бһ“бһ№бһ„бһҹбһҡбһ»бһ”: бһҒбҹ’бһ“бһ¶бһҸбһҳбһјбһӣ
> - бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһ бһ¶бһүбһҖбһ¶бһҡбһ…бҹӮбһҖбһ…бһ¶бһҷбһ“бҹғбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ: бһҡбһјбһ” scatterplot, бһ”бһ¶бһҡ
> - бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһ бһ¶бһүбһ“бһ·бһ“бҹ’бһ“бһ¶бһҖбһ¶бһҡ: бһҮбһҪбһҡбһҲбһҡ, бһҮбһҪбһҡбһҲбһҡ
> - бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһ бһ¶бһүбһ‘бҹҶбһ“бһ¶бһҖбҹӢбһ‘бҹҶбһ“бһ„бһ“бҹғбһҸбһҳбҹ’бһӣбҹғ: бһҮбһҪбһҡбһҲбһҡ, бһҡбһјбһ” scatterplot, бһҡбҹүбһ»бһ„бһ”бҹҠбһ»бһӣ

бһ”бһҫбһўбҹ’бһ“бһҖбһҳбһ¶бһ“бһҸбһ¶бһҡбһ¶бһ„бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ бһ бһҫбһҷбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҡбһҖбһҳбһҫбһӣбһҗбһ¶бһҸбһҫбһҳбһ¶бһ“бһ”бҹүбһ»бһ“бҹ’бһҳбһ¶бһ“бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ’бһ¶бһҸбһ»бһҠбҹӮбһӣбһ”бһ¶бһ“бһ•бҹ’бһҠбһӣбҹӢ бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһҸбҹ’бһҡбһјбһңбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһҠбҹ„бһҷбһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҸбһҳбҹ’бһӣбҹғбһҡбһ”бһҹбҹӢбһңбһ¶бҹ”

вң… бһҳбһ¶бһ“ 'cheat sheets' бһӣбҹ’бһўбһҳбһҪбһҷбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ Matplotlib [бһ“бҹ…бһ‘бһёбһ“бҹҒбҹҮ](https://matplotlib.org/cheatsheets/cheatsheets.pdf)бҹ”

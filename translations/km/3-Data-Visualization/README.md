### бһҖбһ¶бһҡбһҹбһҡбһҹбһҫбһҡ

бһҳбҹҒбһҡбҹҖбһ“бһ”бһ„бҹ’бһ бһ¶бһүбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҹбһҡбһҹбҹҒбһҡбһҮбһ¶бһҳбһҪбһҷрҹҢё бһҠбҹ„бһҷ [Jen Looper](https://twitter.com/jenlooper), [Jasleen Sondhi](https://github.com/jasleen101010) бһ“бһ·бһ„ [Vidushi Gupta](https://github.com/Vidushi-Gupta)бҹ”

рҹҚҜ бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһ•бһӣбһ·бһҸбһ‘бһ№бһҖбһғбҹ’бһҳбһ»бҹҶбһ“бҹ…бһўбһ¶бһҳбҹҒбһҡбһ·бһҖбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҷбһҖбһ–бһёбһӮбһҳбҹ’бһҡбҹ„бһ„бһҡбһ”бһҹбҹӢ Jessica Li бһ“бҹ…бһӣбһҫ [Kaggle](https://www.kaggle.com/jessicali9530/honey-production)бҹ” [бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ](https://usda.library.cornell.edu/concern/publications/rn301137d) бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ‘бһ‘бһҪбһӣбһ–бһё [бһҖбһ¶бһҡбһ·бһҷбһ¶бһӣбҹҗбһҷбһҡбһҠбҹ’бһӢбһўбһ¶бһҳбҹҒбһҡбһ·бһҖбһ•бҹ’бһ“бҹӮбһҖбһҖбһҹбһ·бһҖбһҳбҹ’бһҳ](https://www.nass.usda.gov/About_NASS/index.php)бҹ”

рҹҚ„ бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ•бҹ’бһҹбһ·бһҸбһҖбҹҸбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҷбһҖбһ–бһё [Kaggle](https://www.kaggle.com/hatterasdunton/mushroom-classification-updated-dataset) бһҠбҹӮбһӣбһ”бһ¶бһ“бһҖбҹӮбһ”бҹ’бһҡбҹӮбһҠбҹ„бһҷ Hatteras Duntonбҹ” бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ“бҹҒбҹҮбһҡбһҪбһҳбһҳбһ¶бһ“бһҖбһ¶бһҡбһ–бһ·бһ–бһҺбҹҢбһ“бһ¶бһўбҹҶбһ–бһёбһӮбҹҶбһҡбһјбһҖбһ¶бһӣаҘҚаӨӘаӨЁаӨҝаӨ•бһ‘бһ¶бһҖбҹӢбһ‘бһ„бһ“бһ№бһ„бһ”бҹҶбһҡбҹӮбһ”бҹҶбһҡбһҪбһӣ бҹўбҹЈ бһ”бҹ’бһҡбһ—бҹҒбһ‘бһ•бҹ’бһҹбһ·бһҸбһҠбҹӮбһӣбһҳбһ¶бһ“бһҡбһ¶бһ„бһҖбҹ’бһҡбһ…бһҖбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбҹ’бһҡбһ»бһҳ Agaricus бһ“бһ·бһ„ Lepiotaбҹ” бһ•бҹ’бһҹбһ·бһҸбһӮбһ¶бһҡбһ–бһёбһ–бһ бһ»бһҳбһ»бһҒбһҳбһ¶бһҸбҹӢ Audubon Society бһ…бҹ’бһҡбһҫбһ“бһ”бһ¶бһ“бһ…бһ„бһҖбҹ’бһҡбһ„бһ“бҹ…бһҶбҹ’бһ“бһ¶бҹҶ бҹЎбҹ©бҹЁбҹЎбҹ” бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ“бҹҒбҹҮбһ”бһ¶бһ“бһўбҹҶбһҺбҹ„бһҷбһ‘бҹ… UCI ML 27 бһҶбҹ’бһ“бһ¶бҹҶ бҹЎбҹ©бҹЁбҹ§бҹ”

рҹҰҶ бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бһҖбҹ’бһҹбһё Minnesota бһҳбһҖбһ–бһё [Kaggle](https://www.kaggle.com/hannahcollins/minnesota-birds) бһҠбҹӮбһӣбһ”бһ¶бһ“бһҷбһҖбһ–бһё [Wikipedia](https://en.wikipedia.org/wiki/List_of_birds_of_Minnesota) бһҠбҹ„бһҷ Hannah Collinsбҹ”

бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһ“бҹҒбҹҮбһҳбһ¶бһ“бһўбһ¶бһҮбҹ’бһүбһ¶бһ”бһҺбҹ’бһҺбһҮбһ¶ [CC0: Creative Commons](https://creativecommons.org/publicdomain/zero/1.0/).

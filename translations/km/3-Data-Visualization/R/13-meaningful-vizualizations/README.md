бһҳбһ“бһ»бһҹбҹ’бһҹбһңбһ·бһ‘бҹ’бһҷбһ¶бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҳбһҪбһҷбһҳбһ¶бһ“бһҮбҹҶбһ“бһ¶бһүбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһҳбһҪбһҷбһӮбһәбһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһ”бһ„бҹ’бһҖбһҫбһҸбһҖбһ¶бһҡбһҳбһҫбһӣбһғбһҫбһүбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҠбҹӮбһӣбһҳбһ¶бһ“бһўбһҸбҹ’бһҗбһ“бҹҗбһҷбһҠбҹӮбһӣбһҮбһҪбһҷбһІбҹ’бһҷбһҳбһ¶бһ“бһҖбһ¶бһҡбһҶбҹ’бһӣбһҫбһҷбһҹбҹҶбһҺбһҪбһҡбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһҳбһ¶бһ“бҹ” бһҳбһ»бһ“бһ–бҹҒбһӣбһ”бһ„бҹ’бһ бһ¶бһүбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ бһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһҸбҹӮбһ’бһ¶бһ“бһ¶бһҗбһ¶бһңбһ¶бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҹбһҳбҹ’бһўбһ¶бһҸ бһ“бһ·бһ„бһҡбҹҖбһ”бһ…бҹҶбһҡбһҪбһ…бһ бһҫбһҷ бһҠбһјбһ…бһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһ’бҹ’бһңбһҫбһ“бҹ…бһ…бҹҶбһҺбҹӮбһҖбһҳбһ»бһ“бҹ” бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһҳбһҖ бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһҹбһҳбҹ’бһҡбҹҒбһ…бһ…бһ·бһҸбҹ’бһҸбһ–бһёбһҡбһ”бҹҖбһ”бһӣбҹ’бһўбһ”бҹҶбһ•бһ»бһҸбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ”бһ„бҹ’бһ бһ¶бһүбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбҹ”

бһҖбҹ’бһ“бһ»бһ„бһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮ бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһЎбһҫбһ„бһңбһ·бһүбҹ–

1. бһҡбһ”бҹҖбһ”бһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбһ”бҹ’бһҡбһ—бҹҒбһ‘бһҸбһ¶бһҡбһ¶бһ„бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһң
2. бһҡбһ”бҹҖбһ”бһ…бҹҖбһҹбһңбһ¶бһ„бһҖбһ¶бһҡбһ”бҹ„бһҖбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҸбһ¶бһҡбһ¶бһ„
3. бһҡбһ”бҹҖбһ”бһ”бҹ’бһҡбһҫбһ–бһҺбҹҢ
4. бһҡбһ”бҹҖбһ”бһ”бҹҠбһ·бһ…бһ”бһ“бҹ’бһҸбһ¶бһ”бһҸбһ¶бһҡбһ¶бһ„бһІбҹ’бһҷбһўбһ¶бһ…бһўбһ¶бһ“бһ”бһ¶бһ“бһ„бһ¶бһҷ
5. бһҡбһ”бҹҖбһ”бһҖбһҹбһ¶бһ„бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһҸбһ¶бһҡбһ¶бһ„бһңбһёбһҠбҹҒбһўбһјбһўбһ“бһ»бһңбһҸбҹ’бһҸбһ¬ 3D
6. бһҡбһ”бҹҖбһ”бһҖбһҹбһ¶бһ„бһҖбһ¶бһҡбһҳбһҫбһӣбһғбһҫбһүбһ”бҹӮбһ”бһ…бҹ’бһ“бҹғбһ”бҹ’бһҡбһҢбһ·бһҸ

## бһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбһ”бҹ’бһҡбһ—бҹҒбһ‘бһҸбһ¶бһҡбһ¶бһ„бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһң

бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҳбҹҒбһҡбҹҖбһ“бһҳбһ»бһ“бҹ— бһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһ”бһ„бҹ’бһҖбһҫбһҸбһҖбһ¶бһҡбһҳбһҫбһӣбһғбһҫбһүбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһӮбҹ’бһҡбһ”бҹӢбһ”бҹ’бһҡбһ—бҹҒбһ‘бһҠбҹҸбһӮбһҪбһҡбһұбҹ’бһҷбһ…бһ¶бһ”бҹӢбһўбһ¶бһҡбһҳбҹ’бһҳбһҺбҹҚбһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ Matplotlib бһ“бһ·бһ„ Seaborn бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһӮбһјбһҡбһҹбһ“бҹ’бһӣбһ№бһҖбҹ” бһ‘бһјбһ‘бҹ… бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҮбҹ’бһҡбһҫбһҹбһ”бҹ’бһҡбһ—бҹҒбһ‘бһҸбһ¶бһҡбһ¶бһ„бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһңбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҹбҹҶбһҺбһҪбһҡбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһҖбҹҶбһ–бһ»бһ„бһҹбһҪбһҡбһҸбһ¶бһҳбһҸбһ¶бһҡбһ¶бһ„бһ“бҹҒбҹҮбҹ–

## бһҮбҹҖбһҹбһңбһ¶бһ„бһҖбһ¶бһҡбһ”бҹ„бһҖбһ”бҹ’бһҡбһ¶бһҹбҹӢ

бһ”бһҫбһ‘бҹ„бҹҮбһ”бҹүбһ»бһ“бҹ’бһҸбһ·бһҳбһ“бһ»бһҹбҹ’бһҹбһңбһ·бһ‘бҹ’бһҷбһ¶бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҳбһ¶бһ“бһҖбһ¶бһҡбһ”бҹ’бһҡбһ»бһ„бһ”бҹ’бһҡбһҷбҹҗбһҸбҹ’бһ“бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбһҸбһ¶бһҡбһ¶бһ„бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһңбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһң бһҖбҹҸбһҳбһ¶бһ“бһңбһ·бһ’бһёбһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бһҠбҹӮбһӣбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһўбһ¶бһ…бһ”бһ„бҹ’бһ бһ¶бһүбһҠбҹ„бһҷбһҡбһ”бҹҖбһ”бһҳбһ·бһ“бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһң бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһүбҹ’бһҮбһ¶бһҖбҹӢбһ…бҹҶбһҺбһ»бһ…бһҺбһ¶бһҳбһҪбһҷ бһҮбһ¶бһ…бһҳбҹ’бһ”бһ„бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ”бҹҶбһ—бҹ’бһӣбһәбһ–бһёбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбҹ” бһҳбһ¶бһ“бһ§бһ‘бһ¶бһ бһҡбһҺбҹҢбһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бһ“бҹғбһҸбһ¶бһҡбһ¶бһ„ бһ“бһ·бһ„бһҡбһјбһ”бһ—бһ¶бһ–бһ”бҹ„бһҖбһ”бҹ’бһҡбһ¶бһҹбҹӢ!

бһҸбһ¶бһҡбһ¶бһ„бһҡбһ”бҹҖбһ”бһ…бһҳбҹ’бһӣбҹӮбһҖбһ“бҹҒбҹҮбһ”бһ„бҹ’бһ бһ¶бһүбһҡбһ”бҹҖбһ”бһҠбҹӮбһӣбһўбһ¶бһ…бһ”бҹ’бһҸбһјбһҡбһ—бһ¶бһӮбһҡбһҷбһ”бһ¶бһ“ бһҠбҹ„бһҷбһҳбһ¶бһ“бһ•бһӣбһ”бҹүбҹҮбһ–бһ¶бһӣбҹӢбһӮбһҪбһҡбһІбҹ’бһҷбһҹбһҫбһ…бҹ–

![bad chart 4](../../../../../translated_images/km/bad-chart-4.8872b2b881ffa96c.webp)

бһҖбһ¶бһҡбһ”бҹ’бһҡбҹҖбһ”бһ’бҹҖбһ”бһўбҹ’бһңбһёбһҠбҹӮбһӣбһҳбһ·бһ“бһўбһ¶бһ…бһ”бҹ’бһҡбҹҖбһ”бһ”бһ¶бһ“бһӮбһәбһҮбһ¶бһ‘бһ“бҹ’бһӣбһ№бһҳбһҳбһҪбһҷбһ‘бҹҖбһҸбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ”бҹ„бһҖбһ”бҹ’бһҡбһ¶бһҹбҹӢбҹ” бһҳбһ¶бһ“бһңбҹҒбһ”бһҹбһ¶бһҷбһҳбһҪбһҷбһӮбһҪбһҡбһұбҹ’бһҷбһ…бһ¶бһ”бҹӢбһўбһ¶бһҡбһҳбҹ’бһҳбһҺбҹҚ [wonderful web site](https://tylervigen.com/spurious-correlations) бһҹбҹ’бһҸбһёбһ–бһё 'бһҹбһҳбһ¶бһӮбһҳбһ”бҹӮбһ”бһҖбҹ’бһӣбҹӮбһ„бһҖбҹ’бһӣбһ¶бһҷ'(spurious correlations) бһ”бһ„бҹ’бһ бһ¶бһү 'бһҖбһ¶бһҡбһ–бһ·бһҸ' бһҠбҹӮбһӣбһ–бһ¶бһҖбҹӢбһ–бҹҗбһ“бҹ’бһ’бһҠбһјбһ…бһҮбһ¶бһўбһҸбҹ’бһҡбһ¶бһҖбһ¶бһҡбһ”бҹӮбһҖбһӮбҹ’бһ“бһ¶бһҖбҹ’бһ“бһ»бһ„бһҡбһҠбҹ’бһӢ Maine бһ“бһ·бһ„бһҖбһ¶бһҡбһ”бһҡбһ·бһ…бҹ’бһҶбҹҒбһ‘бһҳбһ¶бһҹбҹҠбһёбһҳбҹүбһ¶бһҡбһёбһ“бҹ” бһҖбҹ’бһҡбһ»бһҳ Reddit бһҖбҹҸбһ”бһ¶бһ“бһ”бҹ’бһҡбһҳбһјбһӣбһ”бҹ’бһҡбһҳбһ¶бһҺ [бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҳбһ·бһ“бһҹбҹ’бһўбһ¶бһҸ](https://www.reddit.com/r/dataisugly/top/?t=all) бһ“бҹғбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ“бҹҒбҹҮбһ•бһ„бһҠбҹӮбһҡбҹ”

бһңбһ¶бһҳбһ¶бһ“бһҹбһ¶бһҡбҹҲбһҹбҹҶбһҒбһ¶бһ“бҹӢбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҷбһӣбҹӢбһҠбһ№бһ„бһ–бһёбһҡбһ”бҹҖбһ”бһҠбҹӮбһӣбһ—бҹ’бһ“бҹӮбһҖбһўбһ¶бһ…бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бҹ„бһҖбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҷбҹүбһ¶бһ„бһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣбһҠбҹ„бһҷбһҸбһ¶бһҡбһ¶бһ„бһ”бҹ„бһҖбһ”бҹ’бһҡбһ¶бһҹбҹӢбҹ” бһ‘бҹ„бҹҮбһ”бһёбһҮбһ¶бһӮбҹ„бһӣбһ”бҹҶбһҺбһ„бһҳбһ“бһ»бһҹбҹ’бһҹбһңбһ·бһ‘бҹ’бһҷбһ¶бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһӣбҹ’бһў бһҖбһ¶бһҡбһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбһ”бҹ’бһҡбһ—бҹҒбһ‘бһҸбһ¶бһҡбһ¶бһ„бһҳбһ·бһ“бһӣбҹ’бһў бһҠбһјбһ…бһҮбһ¶ бһҸбһ¶бһҡбһ¶бһ„бһҶбҹҒбҹҮбһҠбҹӮбһӣбһ”бһ„бҹ’бһ бһ¶бһүбһ”бҹ’бһҡбһ—бҹҒбһ‘бһ…бҹ’бһҡбһҫбһ“бһ–бҹҒбһҖ бһўбһ¶бһ…бһ•бҹ’бһҸбһӣбҹӢбһҖбһ¶бһҡбһ”бҹ„бһҖбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ”бһ¶бһ“бҹ”

## бһ–бһҺбҹҢ

бһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһғбһҫбһүбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҸбһ¶бһҡбһ¶бһ„ 'бһ бһ·бһ„бҹ’бһҹбһ¶бһҖбһ¶бҹҶбһ—бҹ’бһӣбһҫбһ„ Florida' бһҒбһ¶бһ„бһӣбһҫ бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ–бһҺбҹҢбһўбһ¶бһ…бһ•бҹ’бһҸбһӣбҹӢбһҹбҹ’бһҡбһ‘бһ¶бһ”бҹӢбһўбһҸбҹ’бһҗбһ“бҹҗбһҷбһ”бһ“бҹ’бһҗбҹӮбһҳбһ‘бҹ…бһӣбһҫбһҸбһ¶бһҡбһ¶бһ„ бһҡбҹҶбһ–бһ¶бһҖбҹӢбһҮбһ¶бһ–бһ·бһҹбҹҒбһҹбһ“бһјбһңбһҸбһ¶бһҡбһ¶бһ„бһҠбҹӮбһӣбһҳбһ·бһ“бһ”бһ¶бһ“бһҡбһ…бһ“бһ¶бһЎбһҫбһ„бһҠбҹ„бһҷбһ”бһҺбҹ’бһҺбһ¶бһӣбҹҗбһҷбһҠбһјбһ…бһҮбһ¶ ggplot2 бһ“бһ·бһ„ RColorBrewer бһҠбҹӮбһӣбһҳбһ¶бһ“бһ”бһҺбҹ’бһҺбһ¶бһӣбҹҗбһҷбһ–бһҺбҹҢ бһ“бһ·бһ„бһ•бҹ’бһӣбҹӮбһҲбһҫбһ“бһ¶бһ“бһ¶бһҠбҹӮбһӣбһ”бһ¶бһ“бһўбһ“бһ»бһҳбҹҗбһҸбһҮбһ¶бһ”бҹ’бһҡбһ…бһ¶бҹҶбҹ” бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһўбҹ’бһ“бһҖбһҖбҹҶбһ–бһ»бһ„бһ”бһ„бҹ’бһҖбһҫбһҸбһҸбһ¶бһҡбһ¶бһ„бһҠбҹ„бһҷбһҠбҹғ бһҹбһјбһҳбһҹбһ·бһҖбҹ’бһҹбһ¶бһ–бһё [бһ‘бҹ’бһҡбһ№бһҹбҹ’бһҸбһёбһ–бһҺбҹҢ](https://colormatters.com/color-and-design/basic-color-theory) бһ”бһ“бҹ’бһҗбҹӮбһҳбҹ”

> вң… бһҹбһјбһҳбһҷбһҖбһ…бһ·бһҸбҹ’бһҸбһ‘бһ»бһҖбһҠбһ¶бһҖбҹӢ бһ–бҹҒбһӣбһҡбһ…бһ“бһ¶бһҸбһ¶бһҡбһ¶бһ„бһҗбһ¶ бһ…бҹҶбһҺбһ»бһ…бһ…бһјбһӣбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҸбҹ’бһҡбһјбһңбһҳбһ¶бһ“бһҹбһ¶бһҡбҹҲбһҹбҹҶбһҒбһ¶бһ“бҹӢбҹ” бһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҒбҹ’бһӣбҹҮбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһ бҹӮбһӣбһҮбһ¶вҖӢбһҳбһ¶бһ“бһ—бһ¶бһ–бһҳбһ·бһ“бһўбһ¶бһ…бһҳбһҫбһӣбһғбһҫбһүбһ–бһҺбҹҢ вҖ“ бһҸбһҫбһҸбһ¶бһҡбһ¶бһ„бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ”бһ„бҹ’бһ бһ¶бһүбһӣбҹ’бһўбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбҹ’бһ“бһҖбһҠбҹӮбһӣбһҳбһ¶бһ“бһ—бһ¶бһ–бһҳбһ·бһ“бһғбһҫбһүбһ–бһҺбҹҢ?

бһ”бҹ’бһҡбһҷбҹҗбһҸбҹ’бһ“бһ–бҹҒбһӣбһҮбҹ’бһҡбһҫбһҹбһ–бһҺбҹҢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҸбһ¶бһҡбһ¶бһ„бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ бһ–бһёбһ–бҹ’бһҡбҹ„бҹҮбһ–бһҺбҹҢбһўбһ¶бһ…бһ”бһүбҹ’бһҮбһ¶бһҖбҹӢбһўбһҸбҹ’бһҗбһ“бҹҗбһҷбһҳбһҪбһҷбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһ бҹӮбһӣбһҮбһ¶бһҳбһ·бһ“бһ…бһ„бҹӢбһ”бһ¶бһ“бһ‘бҹҒбҹ” 'бһ“бһ¶бһҡбһёбһ•бҹ’бһҖбһ¶бһҲбһјбһҖ' бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҸбһ¶бһҡбһ¶бһ„ 'бһҡбһҷбҹҲбһ”бһ¶бһҸ' бһҒбһ¶бһ„бһӣбһҫбһ”бһүбҹ’бһҮбһ¶бһҖбҹӢбһўбһҸбҹ’бһҗбһ“бҹҗбһҷ 'бһҹбҹ’бһҸбҹ’бһҡбһё' бһҳбһҪбһҷбһҠбҹӮбһӣбһ”бһ“бҹ’бһҗбҹӮбһҳбһ—бһ¶бһ–бһўбһҹбҹ’бһ…бһ¶бһҡбҹ’бһҷбһ‘бҹ…бһҸбһ¶бһҡбһ¶бһ„бҹ”

бһҒбһҺбҹҲбһ–бҹҒбһӣбһҠбҹӮбһӣ [бһўбһҸбҹ’бһҗбһ“бҹҗбһҷбһ–бһҺбҹҢ](https://colormatters.com/color-symbolism/the-meanings-of-colors) бһ”бҹ’бһҡбһ бҹӮбһӣбһҮбһ¶бһ–бһёбһҡбһ–бһҺбҹҢбһӮбҹ’бһ“бһ¶бһ“бҹ…бһҸбҹҶбһ”бһ“бҹӢбһ•бҹ’бһҹбҹҒбһ„бҹ—бһ“бҹғбһ–бһ·бһ—бһ–бһӣбҹ„бһҖ бһ бһҫбһҷбһҳбһ¶бһ“бһҖбһ¶бһҡбһ”бҹ’бһҡбҹӮбһ”бҹ’бһҡбһҪбһӣбһҖбҹ’бһ“бһ»бһ„бһўбһҸбҹ’бһҗбһ“бҹҗбһҷбһ‘бҹ…бһҸбһ¶бһҳбһҹбҹ’бһҡбһ‘бһ¶бһ”бҹӢбһ–бһҺбҹҢбһҡбһ”бһҹбҹӢбһ–бһҪбһҖбһңбһ¶бҹ” бһҮбһ¶бһ—бһ¶бһӮбһ…бҹ’бһҡбһҫбһ“ бһўбһҸбҹ’бһҗбһ“бҹҗбһҷбһ–бһҺбҹҢбһҡбһҪбһҳбһҳбһ¶бһ“бҹ–

## бһҖбһ¶бһҡбһ”бҹҠбһ·бһ…бһ”бһ“бҹ’бһҸбһ¶бһҸбһ¶бһҡбһ¶бһ„бһІбҹ’бһҷбһўбһ¶бһ…бһўбһ¶бһ“бһ”бһ¶бһ“бһ„бһ¶бһҷ

бһҸбһ¶бһҡбһ¶бһ„бһҳбһ·бһ“бһҳбһ¶бһ“бһўбһҸбҹ’бһҗбһ“бҹҗбһҷбһ‘бҹҒ бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһңбһ¶бһҳбһ·бһ“бһўбһ¶бһ…бһўбһ¶бһ“бһ”бһ¶бһ“! бһҹбһјбһҳбһҷбһҖбһ–бҹҒбһӣбһҳбһҫбһӣбһ‘бҹ…бһҖбһ¶бһ“бҹӢбһҖбһ¶бһҡбһ”бҹҠбһ·бһ…бһ”бһ“бҹ’бһҸбһ¶бһ”бҹҶбһ•бһ»бһҸбһ‘бһ‘бһ№бһ„ бһ“бһ·бһ„бһҖбһҳбҹ’бһ–бһҹбҹӢбһҡбһ”бһҹбҹӢбһҸбһ¶бһҡбһ¶бһ„бһІбҹ’бһҷбһҹбһҳбһҹбҹ’бһҡбһ”бһҮбһ¶бһҳбһҪбһҷбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбҹ” бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһўбһҗбҹҒбһҡбһҳбһҪбһҷ (бһҠбһјбһ…бһҮбһ¶ бһҡбһҠбҹ’бһӢбһ‘бһ¶бҹҶбһ„ 50) бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһ„бҹ’бһ бһ¶бһү бһҹбһјбһҳбһ”бһ„бҹ’бһ бһ¶бһүбһңбһ¶бһЎбһҫбһ„бһӣбһҫ Y axis бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһўбһ¶бһ…бһ’бҹ’бһңбһҫбһ”бһ¶бһ“ бһҠбһҫбһҳбҹ’бһ”бһёбһ…бҹҖбһҹбһңбһ¶бһ„бһҖбһ¶бһҡбһҹбҹ’бһ‘бһјбһҸбһ‘бһ№бһҖбһҠбҹғбһҳбҹ’бһҒбһ¶бһ„бҹ”

бһҹбһҡбһҹбҹҒбһҡбһҲбҹ’бһҳбҹ„бҹҮбһўбһҖбҹ’бһҒбһҡбһ¶бһңбһҡбһҺбҹҲбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ, бһ•бҹ’бһҸбһӣбҹӢбһ‘бҹ’бһҡбһ„бҹ’бһңбһ„бҹӢбһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһ…бһ¶бҹҶбһ”бһ¶бһ…бҹӢ бһ бһҫбһҷбһ•бҹ’бһҸбһӣбҹӢбһ§бһ”бһҖбһҡбһҺбҹҚбһҮбҹҶбһ“бһҪбһҷбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһҷбһӣбҹӢбһҠбһ№бһ„бһӣбҹ’бһўбһ”бҹ’бһҡбһҹбһҫбһҡбһҮбһ¶бһ„бһ“бҹҒбҹҮбҹ”

бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһӮбһәбһўбһҸбҹ’бһҗбһ”бһ‘бһ бһҫбһҷбһңбһ¶бһҷбҹүбһ¶бһ„бһҡбҹҶбһ–бҹҒбһ…бһӣбһҫ X axis бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҳбҹҖбһӣбһўбһҸбҹ’бһҗбһ”бһ‘бһІбҹ’бһҷбһҳбһ¶бһ“бһҖбҹ„бһ„бһҳбһҪбһҷбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбһ¶бһ“бһ„бһ¶бһҷбҹ” [plot3D](https://cran.r-project.org/web/packages/plot3D/index.html) бһ•бҹ’бһҸбһӣбҹӢбһҮбһјбһ“бһ“бһјбһңбһ—бһ¶бһ–бһҹбһҳбһ бһҸбҹ’бһҗбһ—бһ¶бһ–бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһӮбһјбһҡ 3d бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһӮбһ¶бҹҶбһ‘бҹ’бһҡбһңбһ¶бҹ” бһҖбһ¶бһҡбһҳбһҫбһӣбһғбһҫбһүбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҹбҹ’бһҳбһ»бһӮбһҹбҹ’бһҳбһ¶бһүбһўбһ¶бһ…бһ”бһ„бҹ’бһҖбһҫбһҸбһ”бһ¶бһ“бһҠбҹ„бһҷбһңбһ¶бҹ”

бһҒбһҺбҹҲбһ–бҹҒбһӣбһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮбһҳбһ·бһ“бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ¶бһ“бҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бһ„бҹ’бһҡбҹҖбһ“бһҮбһҳбҹ’бһҡбһ¶бһӣбһҮбһ¶бһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһўбҹҶбһ–бһёбһ”бһҺбҹ’бһҺбһ¶бһӣбҹҗбһҷбһҳбһҫбһӣбһғбһҫбһүбһҠбҹҸбһҳбһ¶бһ“бһҘбһ‘бҹ’бһ’бһ·бһ–бһӣбһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбһ‘бҹҒ бһҹбһјбһҳбһҹбһ¶бһҖбһӣбҹ’бһ”бһ„ D3 бһҖбҹ’бһ“бһ»бһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё Vue.js бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫбһ”бһҺбҹ’бһҺбһ¶бһӣбҹҗбһҷбһҳбһҪбһҷбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бһ„бҹ’бһ бһ¶бһүбһҖбһ¶бһҡбһҳбһҫбһӣбһғбһҫбһүбһҹбҹҖбһңбһ—бҹ… "Dangerous Liaisons" бһҮбһ¶бһ”бһҺбҹ’бһҠбһ¶бһүбһҹбһ„бҹ’бһӮбһҳбһңбһ·бһӣбһҸбһҪбҹ”

> "Les Liaisons Dangereuses" бһӮбһәбһҮбһ¶бһҡбһҝбһ„бһ“бһ·бһ‘бһ¶бһ“бһҖбҹ’бһ“бһ»бһ„бһҡбһјбһ”бһҳбһ“бҹ’бһҸ epistolary novel, бһ¬бһҮбһ¶бһҡбһҝбһ„бһ“бһ·бһ‘бһ¶бһ“бһҠбҹӮбһӣбһ”бһ„бҹ’бһ бһ¶бһүбһҮбһ¶бһҹбҹҶбһҺбһ»бҹҶбһӣбһ·бһҒбһ·бһҸбһҳбһҪбһҷбҹ” бһ”бһ¶бһ“бһҹбһҡбһҹбҹҒбһҡбһЎбһҫбһ„бһҖбҹ’бһ“бһ»бһ„бһҶбҹ’бһ“бһ¶бҹҶ бҹЎбҹ§бҹЁбҹў бһҠбҹ„бһҷ Choderlos de Laclos бҹ” бһңбһ¶бһҖбһ¶бҹҶбһҖбһ¶бһҸбҹӢбһҡбһҝбһ„бһҡбһ”бһҹбҹӢбһҖбһ¶бһҡбһ…бҹҒбһүбһ…бһӣбһ“бһ¶бһҸбһ¶бһҳбһҹбһ„бҹ’бһӮбһҳбһҘбһҸбһӣбҹ„бһ—бһӣбҹҗбһҖбҹ’бһҒбһ–бһёбһҳбһ“бһ»бһҹбҹ’бһҹбһ…бҹ’бһҡбһҫбһ“бһҖбҹ’бһ“бһ»бһ„бһўбһ¶бһҗбҹҢбһҖбҹҶбһ”бһ¶бҹҶбһ„бһҹбһ„бҹ’бһӮбһҳбһ”бҹӮбһ”бһҡбһ”бһҹбҹӢбһўбһ¶бһҺбһ¶бһ–бҹ’бһҷбһ¶бһ”бһ¶бһӣбһ”бһ¶бһҡбһ¶бҹҶбһ„бһ“бҹ…бһҠбһҫбһҳбһ‘бһё бҹЎбҹЁ бһҹбһҸбһңбһҸбҹ’бһҹ Vicomte de Valmont бһ“бһ·бһ„ Marquise de Merteuilбҹ” бһ–бһҪбһҖбһӮбҹҒбһҹбҹ’бһӣбһ¶бһ”бҹӢбһ“бҹ…бһ…бһ»бһ„бһ”бһүбҹ’бһ…бһ”бҹӢ бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһӮбҹ’бһҳбһ¶бһ“бһҖбһ¶бһҡбһ…бҹ„бһ‘бһҗбһ¶бһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһІбҹ’бһҷбһҳбһ¶бһ“бһҖбһ¶бһҡбһ”бҹүбҹҮбһ–бһ¶бһӣбҹӢбһҹбһ„бҹ’бһӮбһҳбһҷбҹүбһ¶бһ„бһҒбҹ’бһӣбһ¶бҹҶбһ„бһҳбһҪбһҷбҹ” бһҡбһҝбһ„бһ“бҹҒбҹҮбһ”бҹӮбһҖбһҒбҹ’бһӣбһҪбһ“бһҮбһ¶бһҹбҹҶбһҺбһ»бҹҶбһўбһҖбҹ’бһҹбһҡбһҠбҹӮбһӣбһ”бһ¶бһ“бһҹбһҡбһҹбҹҒбһҡбһҠбһӣбҹӢбһҳбһ“бһ»бһҹбҹ’бһҹбһ•бҹ’бһҹбҹҒбһ„бҹ—бһҖбҹ’бһ“бһ»бһ„бһңбһ„бҹӢбһҖбһјбһ“бһҖбҹ’бһҡбһ»бһҳбһҡбһ”бһҹбҹӢбһ–бһҪбһҖбһӮбҹҒ бһҡбҹҖбһ”бһ…бҹҶбһ•бҹӮбһ“бһҖбһ¶бһҡбһҹбһ„бҹ’бһҡбҹ’бһӮбҹ„бҹҮбһ¬бһ”бһ„бҹ’бһҖбһҖбһ¶бһҡбҹҒбһҡбҹүбһ»бһҖбҹ” бһ”бһ„бҹ’бһҖбһҫбһҸбһ—бһ¶бһ–бһҳбһҫбһӣбһғбһҫбһүбһ“бҹғбһӣбһ·бһҒбһ·бһҸбһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбһҠбһҫбһҳбҹ’бһ”бһёбһҡбһҖбһғбһҫбһүбһўбҹ’бһ“бһҖбһҹбҹҶбһҒбһ¶бһ“бҹӢбҹ—бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҡбһҝбһ„ бһҠбҹ„бһҷбһҖбһ¶бһҡбһҳбһҫбһӣбһғбһҫбһүбҹ”

бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһ”бһүбҹ’бһ…бһ”бҹӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһңбҹҒбһ”бһҳбһҪбһҷбһҠбҹӮбһӣбһ”бһ„бҹ’бһ бһ¶бһүбһ‘бһ·бһҠбҹ’бһӢбһ—бһ¶бһ–бһңбһ·бһӣбһҸбһҪбһ”бһҺбҹ’бһҠбһ¶бһүбһҹбһ„бҹ’бһӮбһҳбһ“бҹҒбҹҮбҹ” бһңбһ¶бһ”бҹ’бһҡбһҫбһ”бһҺбҹ’бһҺбһ¶бһӣбҹҗбһҷбһҠбҹӮбһӣбһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸбһЎбһҫбһ„бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһҖбһҫбһҸ [бһҡбһјбһ”бһ—бһ¶бһ–бһ”бһҺбҹ’бһҠбһ¶бһү](https://github.com/emiliorizzo/vue-d3-network) бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ Vue.js бһ“бһ·бһ„ D3бҹ” бһ–бҹҒбһӣбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ‘бһ¶бһүбһҖбҹҶбһҺбһҸбҹӢбһ…бҹҶбһҺбһ»бһ…бһ“бҹ…бһӣбһҫбһўбҹҒбһҖбҹ’бһҡбһ„бҹӢбһҠбһҫбһҳбҹ’бһ”бһёбһҡбҹҶбһҖбһ·бһӣбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ”бһ¶бһ“бҹ”

## бһӮбҹҶбһҡбҹ„бһ„бҹ– бһ”бһ„бҹ’бһҖбһҫбһҸбһҸбһ¶бһҡбһ¶бһ„бһ”бһ„бҹ’бһ бһ¶бһүбһ”бһҺбҹ’бһҠбһ¶бһүбһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ D3.js

> бһҜбһҖбһҹбһ¶бһҡбһ бҹ’бһңбһјбһӣбһҢбҹҗбһҡбһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮбһҳбһ¶бһ“бһ бҹ’бһңбһјбһӣбһҢбҹҗбһҡбһҳбһҪбһҷбһҲбҹ’бһҳбҹ„бҹҮ `solution` бһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҡбһҖбһғбһҫбһүбһӮбҹҶбһҡбҹ„бһ„бһ”бһ¶бһ“бһ”бһүбҹ’бһ…бһ”бҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҷбҹ„бһ„бҹ”

1. бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҸбһ¶бһҳбһҖбһ¶бһҡбһҺбҹӮбһ“бһ¶бҹҶбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҜбһҖбһҹбһ¶бһҡ README.md бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҡбһ»бһҖбҹ’бһҒбһҮбһ¶бһҸбһ·бһ бҹ’бһңбһјбһӣбһҢбҹҗбһҡбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳ бһҡбһҪбһ…бһ”бҹ’бһҡбһ¶бһҖбһҠбһҗбһ¶бһўбҹ’бһ“бһҖбһҳбһ¶бһ“ NPM бһ“бһ·бһ„ Node.js бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһӣбһҫбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ бһҳбһ»бһ“бһ“бһ№бһ„бһҸбҹҶбһЎбһҫбһ„бһ•бҹ’бһ“бҹӮбһҖбһ–бһ№бһ„бһ•бҹ’бһўбҹӮбһҖбһ“бҹғбһӮбҹҶбһҡбҹ„бһ„бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ”

2. бһ”бһҫбһҖбһ бҹ’бһңбһјбһӣбһҢбҹҗбһҡбһҠбһјбһ…бһҮбһ¶ `starter/src` бҹ” бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһҡбһҖбһғбһҫбһүбһ бҹ’бһңбһјбһӣбһҢбҹҗбһҡбһҳбһҪбһҷбһҲбҹ’бһҳбҹ„бҹҮ `assets` бһҠбҹӮбһӣбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ“бҹ„бҹҮбһҳбһ¶бһ“бһҜбһҖбһҹбһ¶бһҡ .json бһҠбҹӮбһӣбһҳбһ¶бһ“бһӣбһ·бһҒбһ·бһҸбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһ–бһёбһҡбһҝбһ„бһ“бһ·бһ‘бһ¶бһ“ бһ…бҹҶбһ“бһҪбһ“бһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢ бһ бһҫбһҷбһҳбһ¶бһ“бһҖбһ¶бһҡпҝҪANNOTATED 'to' бһ“бһ·бһ„ 'from'бҹ”

3. бһ”бһүбҹ’бһ…бһ”бҹӢбһҖбһјбһҠбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ `components/Nodes.vue` бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһҫбһҖбһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһҖбһ¶бһҡбһҳбһҫбһӣбһғбһҫбһүбҹ” бһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһҳбҹҒбһ’бһҠбҹ’бһӢ called `createLinks()` бһ бһҫбһҷбһ”бһ“бҹ’бһҗбҹӮбһҳбһӣбҹ’бһ”бһ„ nested loop бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбһ“бҹҒбҹҮбҹ”

бһҡбһ»бһ„бһҡбһҝбһ„бһҖбҹ’бһ“бһ»бһ„бһңбһҸбҹ’бһҗбһ» .json бһҠбһҫбһҳбҹ’бһ”бһёбһ…бһ¶бһ”бҹӢбһҷбһҖбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ 'to' бһ“бһ·бһ„ 'from' бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһӣбһ·бһҒбһ·бһҸ бһ бһҫбһҷбһ”бһ„бҹ’бһҖбһҫбһҸбһўбһ¶бһ”бҹӢбһўject `links` бһІбҹ’бһҷбһ”бһҺбҹ’бһҺбһ¶бһӣбҹҗбһҷбһңбһ·бһ…бһ·бһҸбҹ’бһҡбһҸбҹ’бһҡбһјбһңбһ”бһҡбһ·бһ—бҹ„бһӮбһңбһ¶бҹ–

```javascript
//бһңбһ·бһӣбһҮбһ»бҹҶбһ‘бҹҖбһҸбһҸбһ¶бһҳбһўбһҖбҹ’бһҹбһҡ
      let f = 0;
      let t = 0;
      for (var i = 0; i < letters.length; i++) {
          for (var j = 0; j < characters.length; j++) {
              
            if (characters[j] == letters[i].from) {
              f = j;
            }
            if (characters[j] == letters[i].to) {
              t = j;
            }
        }
        this.links.push({ sid: f, tid: t });
      }
  ```
  
бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ–бһё terminal (npm run serve) бһ бһҫбһҷбһҡбһёбһҖбһҡбһ¶бһҷбһҮбһ¶бһҳбһҪбһҷбһҖбһ¶бһҡбһҳбһҫбһӣбһғбһҫбһү!

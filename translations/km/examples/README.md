## рҹҺҜ бһҡбһ”бҹҖбһ”бһ”бҹ’бһҡбһҫбһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮ

1. **бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһ–бһёбһҠбҹҶбһ”бһјбһ„**бҹ– бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһӣбҹҒбһҒбһҡбҹҖбһ”бһЎбһҫбһ„бһҸбһ¶бһҳбһҖбһҳбҹ’бһҡбһ·бһҸбһ—бһ¶бһ–бһҒбһ»бһҹбһ”бҹ’бһӣбҹӮбһҖбҹ” бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһҮбһ¶бһҳбһҪбһҷ `01_hello_world_data_science.py` бһ бһҫбһҷбһ”бһ“бҹ’бһҸбһ‘бҹ…бһҳбһ»бһҒбҹ”  

2. **бһўбһ¶бһ“бһҖбҹҶбһҺбһҸбҹӢбһҹбһҳбҹ’бһӮбһ¶бһӣбҹӢ**бҹ– бһҜбһҖбһҹбһ¶бһҡбһ“бһёбһҳбһҪбһҷбҹ—бһҳбһ¶бһ“бһҖбҹҶбһҺбһҸбҹӢбһҹбһҳбҹ’бһӮбһ¶бһӣбҹӢбһӣбһҳбҹ’бһўбһ·бһҸбһ–бһ“бҹ’бһҷбһӣбҹӢбһ–бһёбһўбҹ’бһңбһёбһҠбҹӮбһӣбһҖбһјбһҠбһ’бҹ’бһңбһҫ бһ“бһ·бһ„бһ бҹҒбһҸбһ»бһўбҹ’бһңбһёбһ”бһ¶бһ“бһ’бҹ’бһңбһҫбҹ” бһҹбһјбһҳбһўбһ¶бһ“бһҷбҹүбһ¶бһ„бһҷбһҖбһ…бһ·бһҸбҹ’бһҸбһ‘бһ»бһҖбһҠбһ¶бһҖбҹӢ!  

3. **бһҹбһ¶бһҖбһӣбҹ’бһ”бһ„**бҹ– бһ–бҹ’бһҷбһ¶бһҷбһ¶бһҳбһҖбҹӮбһ”бҹ’бһҡбҹӮбһҖбһјбһҠбҹ” бһҳбһ¶бһ“бһўбҹ’бһңбһёбһҖбһҫбһҸбһЎбһҫбһ„бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһўбҹ’бһ“бһҖбһ”бҹ’бһҠбһјбһҡбһҸбһҳбҹ’бһӣбҹғбһҳбһҪбһҷ? бһ”бҹӮбһҖбһ”бһ¶бһҖбҹӢ бһ бһҫбһҷбһҮбһҪбһҹбһҮбһ»бһӣбһңбһ¶ - бһ“бҹ„бҹҮбһҮбһ¶бһҡбһ”бҹҖбһ”бһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһҡбҹҖбһ“!  

4. **бһҡбһҸбҹӢбһҖбһјбһҠ**бҹ– бһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһ“бһёбһҳбһҪбһҷбҹ— бһ бһҫбһҷбһҹбһ„бҹ’бһҖбҹҒбһҸбһӣбһ‘бҹ’бһ’бһ•бһӣбҹ” бһ”бҹ’бһҡбҹҖбһ”бһ’бҹҖбһ”бһңбһ¶бһҮбһ¶бһҳбһҪбһҷбһўбҹ’бһңбһёбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһҡбҹҶбһ–бһ№бһ„бһ‘бһ»бһҖбҹ”  

5. **бһ–бһ„бҹ’бһҡбһёбһҖбһңбһ¶**бҹ– бһ–бҹҒбһӣбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһҷбһӣбҹӢбһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһҳбһҪбһҷ бһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһ–бһ„бҹ’бһҡбһёбһҖбһңбһ¶ бһҮбһ¶бһҳбһҪбһҷбһӮбҹҶбһ“бһ·бһҸбһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҒбҹ’бһӣбһҪбһ“бҹ”  

## рҹ’Ў бһҷбҹ„бһ”бһӣбҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбҹ’бһ“бһҖбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳ

- **бһңбһ¶бһҷбһҖбһјбһҠбһҠбҹ„бһҷбһҒбҹ’бһӣбһҪбһ“бһҜбһ„**бҹ– бһҖбһ»бҹҶбһӮбҹ’бһҡбһ¶бһ“бҹӢбһҸбҹӮбһ…бһҳбҹ’бһӣбһ„бһ”бһ·бһ‘бһ”бһ·бһ‘бҹ” бһҖбһ¶бһҡбһңбһ¶бһҷбһҮбһҪбһҷбһўбҹ’бһ“бһҖбһҡбҹҖбһ“ бһ“бһ·бһ„бһ…бһ„бһ…бһ¶бҹҶ  
- **бһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһҳбҹҒбһҡбҹҖбһ“бһҠбҹӮбһӣбһҳбһ·бһ“бһҹбҹ’бһӮбһ¶бһӣбҹӢ**бҹ– бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһўбҹ’бһ“бһҖбһғбһҫбһүбһўбҹ’бһңбһёбһҳбһҪбһҷбһҠбҹӮбһӣбһҳбһ·бһ“бһҷбһӣбҹӢ бһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһңбһ¶бһ–бһёбһўбҹҠбһ·бһ“бһ’бһәбһҺбҹӮбһҸ бһ¬бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҳбҹҒбһҡбҹҖбһ“бһҹбҹҶбһҒбһ¶бһ“бҹӢбҹ—  
- **бһҹбһҪбһҡбһ–бһ·бһ—бһ¶бһҖбҹ’бһҹбһ¶**бҹ– бһ…бһјбһӣбһҡбһҪбһҳбһҖбҹ’бһ“бһ»бһ„ [бһңбҹҒбһ‘бһ·бһҖбһ¶бһ–бһ·бһ—бһ¶бһҖбҹ’бһҹбһ¶](https://github.com/microsoft/Data-Science-For-Beginners/discussions) бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҮбҹҶбһ“бһҪбһҷ  
- **бһ бҹ’бһңбһ№бһҖбһ бҹ’бһңбһәбһ“бһҮбһ¶бһ”бҹ’бһҡбһ…бһ¶бҹҶ**бҹ– бһ–бҹ’бһҷбһ¶бһҷбһ¶бһҳбһҖбһјбһҠбһ”бҹ’бһҡбһ бҹӮбһӣбһҸбһ·бһ…бҹ—бһҡбҹҖбһ„бһҡбһ¶бһӣбҹӢбһҗбҹ’бһ„бҹғ бһҮбһ¶бһҮбһ¶бһ„бһ–бҹҒбһӣбһңбҹҒбһӣбһ¶бһ’бҹҶбһ’бҹҒбһ„бһҳбһҪбһҷбһ—бһ¶бһӮбһ…бҹ’бһҡбҹҒбһёбһ“бһҖбҹ’бһ“бһ»бһ„бһҳбһҪбһҷбһҹбһ”бҹ’бһҸбһ¶бһ бҹҚ

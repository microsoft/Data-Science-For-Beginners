# Bangun Visualisasi Kustom Anda Sendiri

## Instruksi

Gunakan contoh kode dalam proyek ini untuk membuat sebuah jejaring sosial, dan buat data tiruan dari interaksi sosial Anda sendiri. Anda bisa memetakan penggunaan media sosial Anda atau membuat diagram anggota keluarga Anda. Buatlah aplikasi web yang menarik yang menampilkan visualisasi unik dari sebuah jejaring sosial.

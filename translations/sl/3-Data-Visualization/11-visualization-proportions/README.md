## Krogi!

Malce bolj vizualno zanimiv tabelni diagram je kro≈æni diagram, ki je tabelni diagram z luknjo na sredini. Poglejmo na≈°e podatke s to metodo.

```python
    'amount': capcolor['class']
     }

    FigureClass = Waffle,
    rows = 100,
    values = df.amount,
    labels = list(df.color),
    figsize = (30,30),
    colors=["brown", "tan", "maroon", "green", "pink", "purple", "red", "whitesmoke", "yellow"],
)
```

# 評估表單數據

一位客戶正在測試一個[小型表單](../../../../2-Working-With-Data/08-data-preparation/index.html)，以收集一些關於其客戶群的基本數據。他們將測試結果帶給你，希望你能驗證所收集的數據。你可以在瀏覽器中打開 `index.html` 頁面查看表單。

## 指導

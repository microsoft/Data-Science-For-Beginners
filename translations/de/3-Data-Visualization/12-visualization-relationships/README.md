## Dual-Line-Diagramme

Versuchen Sie ein Multiline-Diagramm, indem Sie zwei Liniendiagramme √ºbereinander legen, Seaborns 'despine' verwenden, um die oberen und rechten Achsen zu entfernen, und `ax.twinx` [abgeleitet von Matplotlib](https://matplotlib.org/stable/api/_as_gen/matplotlib.axes.Axes.twinx.html) verwenden.

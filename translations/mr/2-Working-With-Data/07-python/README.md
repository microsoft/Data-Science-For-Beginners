**Grouping** рдЕрдиреЗрдХрджрд╛ Excel рдордзреАрд▓ *pivot tables* рд╕рд╛рд░рдЦрд╛ рдкрд░рд┐рдгрд╛рдо рдорд┐рд│рд╡рдгреНрдпрд╛рд╕рд╛рдареА рд╡рд╛рдкрд░рд▓реЗ рдЬрд╛рддреЗ. рд╕рдордЬрд╛ рдЖрдкрд▓реНрдпрд╛рд▓рд╛ `LenB` рдЪреНрдпрд╛ рдкреНрд░рддреНрдпреЗрдХ рджрд┐рд▓реЗрд▓реНрдпрд╛ рд╕рдВрдЦреНрдпреЗрд╕рд╛рдареА рдХреЙрд▓рдо `A` рдЪреЗ mean рдореВрд▓реНрдп рдореЛрдЬрд╛рдпрдЪреЗ рдЖрд╣реЗ. рдордЧ рдЖрдкрдг `LenB` рджреНрд╡рд╛рд░реЗ рдЖрдкрд▓рд╛ DataFrame рдЧрдЯрдмрджреНрдз рдХрд░реВ рд╢рдХрддреЛ рдЖрдгрд┐ `mean` рдХреЙрд▓ рдХрд░реВ рд╢рдХрддреЛ:
```python

```

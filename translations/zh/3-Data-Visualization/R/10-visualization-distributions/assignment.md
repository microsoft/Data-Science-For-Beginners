# 应用你的技能

## 说明

到目前为止，你已经使用了明尼苏达州鸟类数据集，探索了有关鸟类数量和种群密度的信息。现在，尝试使用一个不同的数据集来练习这些技术，可以从 [Kaggle](https://www.kaggle.com/) 获取数据集。编写一个 R 脚本，讲述关于这个数据集的故事，并确保在分析中使用直方图。

## 评分标准

# Check Di Dataset

One client don come meet una team make una help dem check how taxi customer dey spend money for New York City during different season.

Dem wan know: **Yellow taxi passengers for New York City dey tip drivers more for winter or summer?**

Una team dey for di [Capturing](Readme.md#Capturing) stage for di Data Science Lifecycle and na you dey in charge to handle di dataset. Dem don give you one notebook and [data](../../../../data/taxi.csv) make you explore.

For dis folder, you go see one [notebook](notebook.ipynb) wey dey use Python to load yellow taxi trip data from di [NYC Taxi & Limousine Commission](https://docs.microsoft.com/en-us/azure/open-datasets/dataset-taxi-yellow?tabs=azureml-opendatasets).
You fit also open di taxi data file for text editor or spreadsheet software like Excel.

## Wetin You Go Do

- Check whether di data for dis dataset fit help answer di question.
- Explore di [NYC Open Data catalog](https://data.cityofnewyork.us/browse?sortBy=most_accessed&utf8=%E2%9C%93). Find another dataset wey fit help answer di client question.
- Write 3 questions wey you go ask di client to understand di problem well well.

Check di [dataset's dictionary](https://www1.nyc.gov/assets/tlc/downloads/pdf/data_dictionary_trip_records_yellow.pdf) and [user guide](https://www1.nyc.gov/assets/tlc/downloads/pdf/trip_record_user_guide.pdf) for more info about di data.

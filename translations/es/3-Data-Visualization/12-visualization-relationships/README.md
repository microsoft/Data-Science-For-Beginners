## Cuadr√≠culas de facetas

Las cuadr√≠culas de facetas toman un aspecto de tu conjunto de datos (en nuestro caso, puedes elegir 'a√±o' para evitar producir demasiadas facetas). Seaborn puede luego crear un gr√°fico para cada una de esas facetas de tus coordenadas x e y elegidas para una comparaci√≥n visual m√°s sencilla. ¬øDestaca el a√±o 2003 en este tipo de comparaci√≥n?

Crea una cuadr√≠cula de facetas continuando con el uso de `relplot` como se recomienda en la [documentaci√≥n de Seaborn](https://seaborn.pydata.org/generated/seaborn.FacetGrid.html?highlight=facetgrid#seaborn.FacetGrid).

## Gr√°ficos de l√≠neas duales

Prueba un gr√°fico de l√≠neas m√∫ltiples superponiendo dos gr√°ficos de l√≠neas uno sobre otro, utilizando `despine` de Seaborn para eliminar las espinas superior y derecha, y usando `ax.twinx` [derivado de Matplotlib](https://matplotlib.org/stable/api/_as_gen/matplotlib.axes.Axes.twinx.html). Twinx permite que un gr√°fico comparta el eje x y muestre dos ejes y. As√≠ que, muestra el rendimiento por colonia y el n√∫mero de colonias, superpuestos:

```python
fig, ax = plt.subplots(figsize=(12,6))
lineplot = sns.lineplot(x=honey['year'], y=honey['numcol'], data=honey, 
                        label = 'Number of bee colonies', legend=False)
sns.despine()
plt.ylabel('# colonies')
plt.title('Honey Production Year over Year');

ax2 = ax.twinx()
lineplot2 = sns.lineplot(x=honey['year'], y=honey['yieldpercol'], ax=ax2, color="r", 
                         label ='Yield per colony', legend=False) 
sns.despine(right=False)
plt.ylabel('colony yield')
ax.figure.legend();
```
![superimposed plots](../../../../3-Data-Visualization/12-visualization-relationships/images/dual-line.png)

Aunque nada salta a la vista alrededor del a√±o 2003, esto nos permite terminar esta lecci√≥n con una nota un poco m√°s feliz: aunque en general hay un n√∫mero decreciente de colonias, el n√∫mero de colonias se est√° estabilizando, incluso si su rendimiento por colonia est√° disminuyendo.

# সহায়তা
## সমস্যা জমা দেওয়া এবং সহায়তা পাওয়ার উপায়  

এই প্রকল্পটি বাগ এবং ফিচার অনুরোধ ট্র্যাক করার জন্য GitHub Issues ব্যবহার করে। নতুন সমস্যা জমা দেওয়ার আগে বিদ্যমান সমস্যাগুলি অনুসন্ধান করুন যাতে ডুপ্লিকেট এড়ানো যায়। নতুন সমস্যার জন্য, আপনার বাগ বা ফিচার অনুরোধ একটি নতুন Issue হিসেবে জমা দিন।

এই প্রকল্প ব্যবহারের জন্য সহায়তা এবং প্রশ্নের ক্ষেত্রে, একটি Issue জমা দিন।

## Microsoft সমর্থন নীতি  

এই রেপোজিটরির জন্য সমর্থন শুধুমাত্র উপরে তালিকাভুক্ত সম্পদগুলির মধ্যে সীমাবদ্ধ।
